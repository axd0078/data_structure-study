# 并查集 (Disjoint Set Union)

并查集的基础实现与优化版本。

## 文件结构

| 文件 | 说明 |
|------|------|
| `dsu.h` | 并查集定义与算法实现 |
| `test_dsu.cpp` | 测试示例 |

## 核心概念

并查集用于管理元素分组，支持：
- **查询**：判断两个元素是否在同一集合
- **合并**：将两个集合合并为一个

## 接口说明

### 基础实现

```c
void init(int s[]);              // 初始化，每个元素独立成集合
int find(int s[], int x);        // 查找根节点（无优化）
void unionSet(int s[], int x, int y);  // 合并集合
```

### 优化实现（推荐）

```c
int findPro(int s[], int x);     // 查找 + 路径压缩
void unionPro(int s[], int x, int y);  // 按秩合并
```

## 优化原理

| 优化 | 说明 |
|------|------|
| 路径压缩 | 查找时将节点直接挂到根节点下 |
| 按秩合并 | 将小集合挂到大集合下，降低树高 |

两者结合后，均摊时间复杂度接近 O(1)。

## 存储规则

- `s[i] < 0`：i 是根节点，`|s[i]|` 为集合大小
- `s[i] >= 0`：i 的父节点是 `s[i]`

## 运行示例

```bash
g++ -std=c++11 test_dsu.cpp -o test && ./test
```
