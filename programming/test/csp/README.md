# CSP 赛题练习

## 模块简介

`programming/test/csp` 目录用于按题号归档 CSP 相关练习题。当前题目覆盖哈希查找、模拟、枚举、蒙特卡洛估算、差分标记、异或校验、位运算统计以及背包优化等类型。

## 文件结构

| 目录 | 说明 |
|------|------|
| `1/` | 基础模拟与查找题 |
| `37/` | 多项式求和类题 |
| `39/` | 概率估算与模板检测题 |
| `40/` | 集合校验题 |
| `41/` | 位运算统计与优化题 |
| `README.md` | 当前目录说明 |

## 题目列表

| 文件 | 题目 | 核心算法 | 复杂度 |
|------|------|----------|--------|
| `1/1.cpp` | 相反数对统计 | 哈希集合查找相反数 | `O(n)` |
| `1/2.cpp` | 窗口点击模拟 | 逆序查找顶层窗口 + 点击后置顶 | `O(nm)` |
| `37/1.cpp` | 偶数点二次函数求和 | 枚举偶数点 + 直接代入 | `O((r-l)/2)` |
| `39/1.cpp` | 蒙特卡洛估算圆周率 | 圆内点统计 | `O(n)` |
| `39/2.cpp` | 阈值模板检测 | 固定模板扫描 + 差分标记 | `O(n^2 + L)` |
| `40/1.cpp` | 集合异或校验 | 子集异或摘要 + 顺序比对 | `O(元素总数)` |
| `40/3.cpp` | 字符矩阵旋转与翻转恢复 | 全局旋转映射 + 局部逆操作模拟 | `O(tZ^2)` |
| `41/1.cpp` | 二进制平衡数统计 | 二进制位计数 | `O(n log V)` |
| `41/2.cpp` | 咖啡时间优化 | 0/1 背包 + 分数贪心 | `O(m×普通任务数 + m×灵活任务数 log 灵活任务数)` |
| `41/3.cpp` | 消息队列区间分配模拟 | `set` / `map` 维护空闲段 + 循环发送 | `O(q log n)` |

## 核心内容

`1/1.cpp` 使用 `unordered_set` 去重后判断 `x` 与 `-x` 是否同时存在，适合练习哈希查找。

`1/2.cpp` 用数组顺序维护窗口层级，每次点击从顶层向下查找命中窗口，并把命中窗口移动到数组末尾表示置顶。

`37/1.cpp` 只枚举区间内的偶数点，把每个点直接代入二次式求和，最后乘以 2 输出结果。

`39/1.cpp` 根据随机点是否落在圆内估算圆周率，核心是用命中比例近似面积比例。

`39/2.cpp` 对每个 5×9 窗口计算合法阈值区间，再用差分数组把所有可能的阈值统一恢复出来。

`40/1.cpp` 先计算标准集合的异或摘要，再和待校验集合同时比较“元素序列是否相同”和“异或值是否相同”。

`40/3.cpp` 把全部操作先记录下来，再逆序恢复字符矩阵；局部旋转、区域翻转都通过当前全局旋转状态映射到原始存储矩阵上完成。

`41/1.cpp` 逐个统计数字二进制表示中 `0` 和 `1` 的数量，判断两者是否相等。

`41/2.cpp` 把任务分成可分割与不可分割两类，分别用分数贪心和 0/1 背包处理，再枚举咖啡分配求最优总耗时。

`41/3.cpp` 用 `map` 维护按左端点索引的空闲区间，用 `set` 维护按长度优先的可分配区间，支持 `new`、`delete`、`send` 三类操作。

## 运行方式

在 `programming/test` 目录下执行：

```bash
g++ -std=c++11 csp/1/1.cpp -o test && ./test
g++ -std=c++11 csp/1/2.cpp -o test && ./test
g++ -std=c++11 csp/37/1.cpp -o test && ./test
g++ -std=c++11 csp/39/1.cpp -o test && ./test
g++ -std=c++11 csp/39/2.cpp -o test && ./test
g++ -std=c++11 csp/40/1.cpp -o test && ./test
g++ -std=c++11 csp/40/3.cpp -o test && ./test
g++ -std=c++11 csp/41/1.cpp -o test && ./test
g++ -std=c++11 csp/41/2.cpp -o test && ./test
g++ -std=c++11 csp/41/3.cpp -o test && ./test
```

在仓库根目录下执行时，需要补全路径：

```bash
g++ -std=c++11 programming/test/csp/39/2.cpp -o test && ./test
```

## 注意事项

- `1/1.cpp` 只统计正数一侧，因此同一组相反数不会重复计数。
- `1/2.cpp` 的窗口边界按闭区间判断，命中后要删除原位置并追加到末尾。
- `39/2.cpp` 的模板大小是固定常数，文档里把扫描复杂度写成 `O(n^2 + L)`。
- `40/1.cpp` 仅靠异或值不能唯一确定集合内容，所以源码同时比较了原下标序列是否一致。
- `40/3.cpp` 里 `rot` 表示当前整体旋转了多少个 90 度，局部操作前后都要通过坐标映射访问真实存储位置。
- `41/2.cpp` 当前实现会对灵活任务多次排序，复盘时重点看“任务拆分类别”和“咖啡容量分配”的思路。
- `41/3.cpp` 分配时依赖“最长度优先、左端点最小优先”的区间选择规则，删除时要注意与左右相邻空闲段合并。
