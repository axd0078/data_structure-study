## 缩写检查

**题目描述**：给定一个以下划线分隔的字符串和其缩写，检查缩写是否正确。缩写由原字符串中每个下划线后的字符组成（包含首字符）。

**输入格式**：
```
t                    // 测试用例数
str1 str2           // 原字符串和缩写（每组）
...
```

**输出格式**：每行输出 "Yes" 或 "No"

**样例**：
```
输入：3
      this_is_a_test tat
      hello_world hw
      a_b_c abc
输出：Yes
      No
      Yes
```

**算法思路**：
- 提取原字符串中首字符和每个下划线后的字符
- 将提取结果与缩写比较
- 时间复杂度：O(n)
