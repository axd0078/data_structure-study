## 三数之和

**题目描述**：给定整数数组，找出所有和为 0 的三元组 `[a, b, c]`，且三元组不重复。

**输入格式**：
```
n
a1 a2 ... an
```

**输出格式**：每行一个三元组，元素从小到大排列

**样例**：
```
输入：6
      -1 0 1 2 -1 -4
输出：-1 -1 2
      -1 0 1
```

**算法思路**：
- 先对数组排序
- 固定第一个数，用双指针找另外两个数
- 跳过重复元素，避免重复解
- 时间复杂度：O(n²)

## 矩阵置零

**题目描述**：给定 m×n 矩阵，如果某个元素为 0，则将其所在行和列所有元素设为 0。

**输入格式**：
```
m n
矩阵元素（m行n列）
```

**输出格式**：处理后的矩阵

**样例**：
```
输入：3 3
      1 1 1
      1 0 1
      1 1 1
输出：1 0 1
      0 0 0
      1 0 1
```

**算法思路**：
- 使用两个标记数组记录哪些行和列需要置零
- 第一遍扫描：找出所有 0 的位置，标记对应行列
- 第二、三遍：根据标记将对应行列置零
- 时间复杂度：O(mn)，空间复杂度：O(m+n)

## 最长递增子序列 (LIS)

**题目描述**：给定整数数组，求最长严格递增子序列的长度。

**输入格式**：
```
n
a1 a2 ... an
```

**输出格式**：一个整数，表示 LIS 长度

**样例**：
```
输入：8
      10 9 2 5 3 7 101 18
输出：4
```

**算法思路**：
- 维护 `tails[]` 数组，`tails[i]` 表示长度为 i+1 的递增子序列的最小末尾元素
- 对每个元素二分查找插入位置，更新或扩展 `tails`
- 时间复杂度：O(n log n)

## 题目10：最短路径

**题目描述**：给定 n 个点 m 条边的有向图，求 1 号点到 n 号点的最短距离。

**输入格式**：
```
n m
u1 v1 w1
u2 v2 w2
...
```

**输出格式**：一个整数，最短距离（不可达输出 -1）

**样例**：
```
输入：3 3
      1 2 2
      1 3 4
      2 3 1
输出：3
```

**算法思路**：
- 使用邻接矩阵存储图
- Dijkstra 算法：每次选取未访问集合中距离最小的点，更新其邻接点
- 时间复杂度：O(n²)

## 题目11：区间合并

**题目描述**：合并所有重叠区间，返回不重叠的区间集合。

**输入格式**：
```
n
l1 r1
l2 r2
...
```

**输出格式**：
```
k
l1' r1'
l2' r2'
...
```

**样例**：
```
输入：4
      1 3
      2 6
      8 10
      15 18
输出：3
      1 6
      8 10
      15 18
```

**算法思路**：
- 按左端点排序
- 遍历区间，若当前区间与前一个重叠则合并，否则加入结果
- 时间复杂度：O(n log n)
