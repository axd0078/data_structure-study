# 第 9 课：查找、二分与可行性判断

## 本课目标

掌握有序数据中的查找，以及答案二分的基本写法。

## lower_bound

```cpp
auto it = lower_bound(a.begin(), a.end(), x);
```

含义：找到第一个大于等于 `x` 的位置。

## upper_bound

```cpp
auto it = upper_bound(a.begin(), a.end(), x);
```

含义：找到第一个大于 `x` 的位置。

统计有序数组中 `x` 出现次数：

```cpp
int cnt = upper_bound(a.begin(), a.end(), x) - lower_bound(a.begin(), a.end(), x);
```

## 手写答案二分

```cpp
int l = 1, r = 1000000;
while (l < r) {
    int mid = l + (r - l) / 2;
    if (check(mid)) {
        r = mid;
    } else {
        l = mid + 1;
    }
}
```

核心不是模板，而是 `check(mid)` 的含义。

## 二分前提

二分一定要有单调性。

如果你说不清：

- 为什么更大一定更容易
- 或者为什么更小一定更容易

那大概率不该用二分。

## 本课练习

### 练习 1：第一次不小于 x

给定升序数组和一个值 `x`，输出第一个大于等于 `x` 的位置。

### 练习 2：统计 x 出现次数

在升序数组中统计某个值出现了多少次。

### 练习 3：最小速度

给定若干任务量和总时间，求满足要求的最小速度。

## 长题面训练方向

这一课适合练一种能力：

> 把复杂叙述压缩成一个 `check(mid)`。

做题时先问自己：

- 枚举答案时，答案越大是越容易还是越难？
- `mid` 固定后，我能不能在线性时间判断它是否可行？
