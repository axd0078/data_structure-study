# 第 9 课：查找、lower_bound 与二分

## 本课目标

掌握有序数组上的查找，以及答案二分的基本模型。

## lower_bound / upper_bound

前提：数组已经升序排序。

```cpp
auto it = lower_bound(a.begin(), a.end(), x);
```

`lower_bound` 找第一个 `>= x` 的位置。

```cpp
auto it = upper_bound(a.begin(), a.end(), x);
```

`upper_bound` 找第一个 `> x` 的位置。

## 答案二分

当题目问“最小的可行值”或“最大的可行值”时，可以考虑答案二分：

```text
如果 x 可行，那么更大的 x 也可行
或
如果 x 可行，那么更小的 x 也可行
```

关键是写出 `check(x)`。

## 本课练习

### 练习 1：第一个不小于 x

输入升序数组和 `q` 次查询。每次查询给出 `x`，输出第一个不小于 `x` 的元素位置，位置从 `1` 开始；如果不存在，输出 `-1`。

测试输入：

```text
5 4
1 3 3 7 10
0
3
8
11
```

测试输出：

```text
1
2
5
-1
```

### 练习 2：统计 x 出现次数

输入升序数组和 `q` 次查询。每次查询给出 `x`，输出 `x` 在数组中出现的次数。

测试输入：

```text
8 4
1 2 2 2 4 4 7 9
2
4
3
9
```

测试输出：

```text
3
2
0
1
```

### 练习 3：最小可行速度

有 `n` 个任务，每个任务工作量为 `a[i]`。如果速度为 `v`，完成第 `i` 个任务需要 `ceil(a[i] / v)` 小时。给定总时间 `h`，求最小速度 `v`，使所有任务能在 `h` 小时内完成。

测试输入：

```text
4 8
3 6 7 11
```

测试输出：

```text
4
```

## 复盘点

- 二分前先确认单调性。
- `lower_bound` 返回迭代器，转下标要减 `a.begin()`。
- `ceil(a / v)` 的整数写法是 `(a + v - 1) / v`。
