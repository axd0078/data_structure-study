# 第 3 课：vector、数组替代与前缀和意识

## 本课目标

学会用 `vector` 存储一维和二维数据，并形成两个判断：

1. 后面还要访问、排序、查询，就存进 `vector`。
2. 只需要计数、求和、最大值，可以边读边处理。

## vector 基础

```cpp
int n;
cin >> n;
vector<int> a(n);
for (int i = 0; i < n; i++) {
    cin >> a[i];
}
```

下标从 `0` 开始，最后一个元素是 `a[n - 1]`。

## 二维 vector

```cpp
int n, m;
cin >> n >> m;
vector<vector<int>> grid(n, vector<int>(m));
```

适合矩阵、表格、网格数据。

## 前缀和意识

如果要多次问区间和，不要每次重新循环求和。常用 1-based 前缀和：

```cpp
vector<long long> prefix(n + 1, 0);
for (int i = 1; i <= n; i++) {
    prefix[i] = prefix[i - 1] + a[i];
}
```

闭区间 `[l, r]` 的和是：

```cpp
prefix[r] - prefix[l - 1]
```

## 本课练习

### 练习 1：最大值位置

输入 `n` 和 `n` 个整数，输出最大值第一次出现的位置。位置从 `1` 开始。

测试输入：

```text
6
1 5 3 5 2 4
```

测试输出：

```text
2
```

### 练习 2：数组反转

输入 `n` 和 `n` 个整数，倒序输出这些整数。

测试输入：

```text
5
1 2 3 4 5
```

测试输出：

```text
5 4 3 2 1
```

### 练习 3：矩阵正数个数

输入 `n`、`m` 和一个 `n x m` 矩阵，统计正数个数。

测试输入：

```text
2 3
1 -2 0
3 4 -5
```

测试输出：

```text
3
```

### 练习 4：区间和入门

输入 `n` 和 `n` 个整数，再输入一个闭区间 `[l, r]`，下标从 `1` 开始，输出该区间元素和。

测试输入：

```text
5
1 2 3 4 5
2 4
```

测试输出：

```text
9
```

## 复盘点

- 计数器必须初始化，例如 `int ans = 0;`。
- 二维循环要确认外层是行还是列。
- 题目说“第几个”通常是 1-based；数组下标通常是 0-based。
