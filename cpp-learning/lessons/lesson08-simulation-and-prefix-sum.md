# 第 8 课：模拟、前缀和与多步骤规则

## 本课目标

掌握两类高频问题：

1. 规则很多、状态会变化的模拟题
2. 频繁求区间和的题

## 模拟题思路

模拟题不是套复杂算法，而是把题目规则翻译成代码：

1. 明确状态变量
2. 按输入顺序更新状态
3. 每次更新后检查边界条件
4. 按题目格式输出

常见状态变量：分数、余额、当前位置、方向、计数器、当前模式。

## 前缀和

如果要频繁求区间和，可以先预处理：

```cpp
vector<int> prefix(n + 1, 0);
for (int i = 1; i <= n; i++) {
    prefix[i] = prefix[i - 1] + a[i];
}
```

区间 `[l, r]` 的和：

```cpp
prefix[r] - prefix[l - 1]
```

## 本课练习

### 练习 1：累计得分

输入 `n` 和 `n` 次得分变化，输出最终得分。

### 练习 2：区间和查询

输入 `n`、`q` 和数组，随后回答多个区间和查询。

### 练习 3：余额模拟

有收入与支出操作，如果支出会让余额小于 0，则该次失败。输出最终余额和失败次数。

## 长题面训练方向

这类题通常会故意把规则写得比较长。做题时不要整段题面一起想，先拆成：

- 初始状态
- 每种操作会改哪些量
- 哪些情况会失败
- 最终要求输出什么

如果规则很多，先写最朴素的状态更新版本，再看有没有必要做前缀和或别的优化。
