# 第 7 课：pair、map、set 与计数建模

## 本课目标

用 STL 做统计、去重、映射和二元数据保存。

## pair

`pair` 用来存两个值：

```cpp
pair<string, int> p = {"alice", 90};
cout << p.first << " " << p.second << endl;
```

常见用途：坐标、姓名和成绩、值和下标。

## map

`map` 是有序映射：

```cpp
map<string, int> cnt;
cnt["apple"]++;
```

遍历时会按 key 从小到大。

## unordered_map

`unordered_map` 平均更快，但不保证顺序。

如果题目要求按字典序输出，用 `map` 更稳。

## set

`set` 自动去重并排序：

```cpp
set<int> s;
s.insert(3);
s.insert(1);
s.insert(3);
```

输出是 `1 3`。

## 本课练习

### 练习 1：整数去重排序

输入 `n` 和 `n` 个整数，输出去重后从小到大的结果。

### 练习 2：单词频次

输入 `n` 和 `n` 个单词，按字典序输出每个单词出现次数。

### 练习 3：成绩最高记录

输入多条姓名和分数记录，输出分数最高的记录；如果分数相同，输出姓名字典序最小的。

## 长题面训练方向

这一课要重点练一种思维：

> 题目真正要你判断的条件，和程序里“看起来相等”的条件，可能不是一回事。

做集合、映射、真假判断类题目时，先分清：

- 真实条件是什么
- 你程序里计算出来的条件是什么

只有这两个条件分离清楚，才不容易误判。
