# 30 天 CSP 复习计划

## 模块简介

`cpp-learning/csp-30days` 用于把 CSP 认证复习拆成 30 天的连续训练计划。每一天都是一份独立的 Markdown 文档，围绕读题流程、T1/T2 保分、典型题型复盘和整套模拟展开。

## 文件结构

| 文件 | 说明 |
|------|------|
| `day1.md` - `day10.md` | 前 10 天：读题流程、T1 稳定得分、基础模拟与统计题回顾 |
| `day11.md` - `day20.md` | 中 10 天：数组、前缀和、排序、二分和常见构造题训练 |
| `day21.md` - `day30.md` | 后 10 天：近届真题、整套模拟和赛后复盘 |

## 核心内容

每一天通常包含三个部分：当天目标、主练真题列表、回顾真题列表。主练部分更偏向第 21 次之后的近届认证题，回顾部分用于补前 20 次中的基础题型和高频套路。

后段文档会逐步加强整套模拟、T3/T4 拆分、赛时取舍和赛后复盘，目标不是堆题量，而是形成稳定的做题流程。

## 使用方式

建议按 `day1.md` 到 `day30.md` 顺序阅读和训练。如果当天时间有限，优先完成“今日目标”里明确标出的主练题，再记录错因和下次改进点。

## 注意事项

- 这些文档是训练索引，不是完整题解；做题时建议配合 `cpp-learning/csp-certification-problems/` 的原题资料使用。
- 如果你已经完成某一阶段训练，可以直接跳到对应天数做复盘，但首次使用更建议按顺序推进。
