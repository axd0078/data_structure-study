# Day 10：差分数组

## ▶ CSP 考情分析

差分数组是 CSP 第 2-4 题中**处理批量区间操作**的核心工具。当你看到"对很多区间做相同的 +1/-1 操作，最后问每个位置的状态"，应该立即想到差分。

**CSP 中考差分的典型信号：**
- "对区间 [l, r] 内所有元素 +1" 反复出现
- 最后问"哪些 k 满足条件"（如 阈值差分 第39次T2）
- "统计覆盖次数"类的批量操作

| 真题 | 考察点 |
|------|--------|
| 阈值差分(第39次T2) | 区间合法标记 + 差分 |
| 出行计划(202203-2) | 差分应用（时间段约束） |
| 灰度直方图(202104-1) | 区间统计（前缀和思想的推广） |

## ▶ 核心技巧

### 差分三步骤

```cpp
// 步骤1：对每个区间 [l, r] 整体 +1
vector<int> diff(n + 2, 0);
diff[l]++;          // 从 l 开始 +1
diff[r + 1]--;      // 从 r+1 开始恢复

// 步骤2：前缀和还原
int cur = 0;
for (int i = 0; i < n; i++) {
    cur += diff[i];
    a[i] = cur;     // a[i] = 位置 i 被多少个区间覆盖
}
```

### 差分 vs 前缀和，别搞混

| 方法 | 方向 | 用途 |
|------|------|------|
| 前缀和 | 累加 → 查询 | 已知原数组，快速求区间和 |
| 差分 | 标记 → 还原 | 已知区间操作，快速求原数组 |

**记忆方法：** 前缀和是"先累积再查"，差分是"先标记再还原"。

## ▶ 真题：模板阈值扫描

**来源：第 39 次认证第 2 题**

n×n 矩阵，用一个 5×9 的二进制模板（1=白格, 0=黑格）扫描每个可能位置。对每个位置：
- 所有白格的值 ≥ k，所有黑格的值 < k
- 得到合法 k 区间 [maxBlack+1, minWhite]

用差分标记所有合法 k，最后输出所有被至少一个模板位置覆盖的 k。

**思路：**
1. 遍历每个 5×9 子矩阵
2. 白格取 min，黑格取 max → 区间 [left, right]
3. `diff[left]++`, `diff[right+1]--`
4. 前缀和还原，输出 cur > 0 的 k

**CSP 考察点：** 这道题的难点不是差分本身，而是"从模板扫描→推出合法区间→用差分批量标记"的完整建模链。

## ▶ 相似真题：出行计划

**试题编号：202203-2（第25次认证第2题）**

n 个出行计划，第 i 个在时刻 t_i 进入，需要 c_i 时间内的核酸证明。有 m 个查询，每个给一个核酸出结果时间 k，问多少个计划满足条件。

**思路：** 每个计划对应一个合法区间 [t_i - c_i + 1, t_i]。用差分标记所有合法时间点。

**CSP 考察点：** 将"时间窗口约束"转换为"合法区间"并用差分——和第39次T2完全相同的思想。
