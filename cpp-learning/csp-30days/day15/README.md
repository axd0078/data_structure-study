# Day 15：位运算

## ▶ CSP 考情分析

位运算在 CSP 中出现的频率越来越高，尤其是 2024 年之后的考试。CSP 对位运算的考察通常不深，但要求你熟练基本操作。

## ▶ 核心技巧

### 基本位运算（必须熟记）

```cpp
x & 1        // 取最低位（判断奇偶）
x >> 1       // 右移 1 位（等价于 x / 2）
x << 1       // 左移 1 位（等价于 x * 2）
x & (x - 1)  // 去掉最低位的 1（判断是否为 2 的幂：x & (x-1) == 0）
x & 7        // 取最低 3 位（等价于 x % 8）
x ^ k        // 按位异或
x | k        // 按位或
n & (1 << i) // 检查第 i 位是否为 1
```

### 统计二进制中 0 和 1 的个数

```cpp
int cnt0 = 0, cnt1 = 0;
while (x > 0) {
    if (x % 2 == 0) cnt0++;
    else cnt1++;
    x /= 2;
}
// 等价写法：
while (x > 0) {
    if (x & 1) cnt1++;
    else cnt0++;
    x >>= 1;
}
```

### 判断 2 的幂

```cpp
bool isPowerOfTwo(int x) {
    return x > 0 && (x & (x - 1)) == 0;
}
```

## ▶ 真题 1：二进制位统计

**来源：第 41 次认证第 1 题**

给定 n 个正整数，统计其中二进制 0 和 1 个数相等的数有多少个。

| 输入样例 | 输出样例 |
|---------|---------|
| 3\n2 3 5 | 1 |

2(10) = 1个0,1个1 ✓；3(11) = 0个0,2个1 ✗；5(101) = 1个0,2个1 ✗

**思路：** 对每个数，除 2 统计 0 和 1 数量，比较是否相等。

## ▶ 真题 2：优秀的拆分

**来源：2020 CSP-J 第 1 题**

判断 n 是否能拆成若干个不同的 2 的幂之和，从大到小输出。

| 输入样例 | 输出样例 |
|---------|---------|
| 6 | 4 2 |

**思路：** 从最高位开始检查 n 的二进制，如果第 i 位为 1 则输出 2^i。

**CSP 考察点：** 十进制和二进制的关系——一个数唯一地对应一组 2 的幂之和。

## ▶ 位运算在 CSP 中的高级应用

**状态压缩：** 用 int 的每一位表示一个元素是否被选。如 0b1011 表示选了第 0,1,3 个元素。

**集合操作：**
```cpp
mask | (1 << i)     // 加入元素 i
mask & ~(1 << i)    // 移除元素 i
mask & (1 << i)     // 检查元素 i 是否在集合中
```
