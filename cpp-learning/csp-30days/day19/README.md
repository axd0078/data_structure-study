## ▶ 核心技巧

### 邻接表建图

```cpp
// vector 存图（CSP 最常用）
int n, m;
vector<int> G[MAXN];  // 无权图
vector<pair<int,int>> GW[MAXN]; // 带权图

for (int i = 0; i < m; i++) {
    int u, v, w;
    cin >> u >> v >> w;
    G[u].push_back(v);
    GW[u].push_back({v, w});
    // 无向图加反向边：
    G[v].push_back(u);
}
```

### DFS 遍历图

```cpp
bool visited[MAXN];

void dfs(int u) {
    visited[u] = true;
    for (int v : G[u]) {
        if (!visited[v])
            dfs(v);
    }
}
```

### 树的直径（两次 DFS）

```cpp
// 从任意点 DFS 找到最远点 p
// 从 p 再 DFS 找到最远点 q
// p 到 q 的距离 = 树的直径
```

## ▶ 真题 1：通信网络

**试题编号：201709-4（第8次认证第4题）**

N 个节点，M 条有向边。问有多少个节点，从它能到达所有节点，且所有节点也能到达它。

**思路：** 对每个节点，分别沿着正向边和反向边做 DFS。正反向都能到达所有节点，该节点才算"中枢"。

**CSP 考察点：** 正反向图 + DFS 全遍历。这是"判断全图连通性"的标准做法。

## ▶ 真题 2：网络延时（树的直径）

**试题编号：201503-4（第4次认证第4题）**

交换机连成树形拓扑，终端电脑只连一台交换机。求任意两台终端电脑之间最远距离。

**思路：** 树的直径问题。第一次 DFS 从根出发找最远点，第二次 DFS 从该点出发找最远距离。

---

## ▶ 真题 3：高速公路（强连通分量）

**试题编号：201509-4（第5次认证第4题）**

N 个城市，M 条有向高速公路。如果两个城市能互相到达，它们属于同一个"便利城市对"。求有多少个城市对。

**思路：** Tarjan 或 Kosaraju 算法求强连通分量。每个 SCC 内如果有 k 个城市，贡献 C(k, 2) 个城市对。
