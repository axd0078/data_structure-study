# Day 6：前缀和

## ▶ CSP 考情分析

前缀和在 CSP 中从 **第 1 题到第 4 题都可能出现**：
- T1-T2 级：简单的区间求和，如 序列查询(202112-1)、坐标变换其二(202306-2)
- T2-T4 级：前缀最值 / 前缀和变体，如 出行计划(202203-2)、梦境巡查(第36次T2)

**CSP 为什么考前缀和？** 因为它把"区间操作 O(n)"变成了"O(1) 查询"。很多题目看似要嵌套循环，其实预处理前缀和/前缀最值后一遍就能过。

**CSP 前缀和题的特征信号：** 题目中出现"连续子段"、"区间和"、"前 i 个的最大/最小值" → 考虑前缀和。

## ▶ 核心技巧

### 前缀和数组

```cpp
vector<int> pre(n + 1);
for (int i = 1; i <= n; i++)
    pre[i] = pre[i - 1] + a[i];
// 区间 [l, r] 和 = pre[r] - pre[l - 1]
```

### 前缀最值（CSP 高频）

```cpp
pre_max[0] = a[0];
for (int i = 1; i <= n; i++)
    pre_max[i] = max(pre_max[i - 1], a[i]);

suf_max[n] = a[n];
for (int i = n - 1; i >= 0; i--)
    suf_max[i] = max(suf_max[i + 1], a[i]);
```

**关键理解：** `pre_max[i]` 存的是"前 i 个元素中的最大值"，这样在查询时不用每次都扫描。

## ▶ 真题 1：序列查询

**试题编号：202112-1（第24次认证第1题）**

给定一个递增序列 A[0..n]（A[0]=0），定义 f(x) = 小于等于 x 的最大 A 元素下标。求 f(0) + f(1) + ... + f(N-1) 的和。

| 输入样例 | 输出样例 |
|---------|---------|
| 3 10\n2 5 8 | 15 |

**思路：** 直接模拟 f(x) 定义。对每个 x，找最大的 i 使 A[i] ≤ x。可以用指针 i 在 x 增长时向后推进——每个 A[i] 的贡献 = (A[i+1] - A[i]) × i。

**CSP 考察点：** 表面是求和，实际是"分段统计"——不需要对每个 x 都从头扫描。

## ▶ 真题 2：坐标变换（其二）

**试题编号：202306-2（第31次认证第2题）**

n 个操作，类型 1 是拉伸 k 倍，类型 2 是旋转 θ 角。m 个查询，问从第 i 个操作执行到第 j 个操作后，初始坐标 (x,y) 变成什么。

**思路：** 维护拉伸的前缀积和旋转角度的前缀和。对于区间 [i, j]：总拉伸 = pre_mul[j] / pre_mul[i-1]，总旋转 = pre_sum[j] - pre_sum[i-1]。

**CSP 考察点：** 前缀思想的推广——前缀和不仅可以是"求和"，还可以是"前缀积"、"前缀角度和"。

## ▶ 真题 3：梦境巡查（综合）

**来源：第 36 次认证第 2 题**

n+1 个区域，a[i] 为移动消耗，b[i] 为到达补给。S[k] = Σ_{0..k} a[i] - Σ_{1..k} b[i]。正常最少初始能量 = max{S[0]..S[n]}。依次假设每个区域出事故（b[i]=0），重新计算最少初始能量。

**思路：**
1. 求前缀和 S[0..n]
2. 求前缀最大值 pre_max[0..n] 和后缀最大值 suf_max[0..n]
3. 第 i 区域事故：ans[i] = max(pre_max[i-1], suf_max[i] + b[i])

**CSP 考察点：** 这道题是前缀和 + 前缀最值的完美结合。如果每次事故都重新扫描整个数组是 O(n²)，用前后缀分解做到 O(n)。
