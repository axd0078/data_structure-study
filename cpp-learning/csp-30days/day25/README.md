# Day 25：树形 DP

## ▶ CSP 考情分析

树形 DP 是 CSP 第 5 题的高频算法，通常与**分组背包、二次扫描、换根**等技术结合。

**CSP 树形 DP 真题：**
| 真题 | 考点 |
|------|------|
| 城市规划(201909-5) | 树形 DP + 分组背包 |
| 食材运输(202012-4) | 二分 + 树形 DP + 状压 DP |
| 网络延时(201503-4) | 树的直径（树形 DP 做法） |
| 阻击(202306-5) | 树形 DP + 线段树 |

**CSP 树形 DP 的特征信号：** 题目给出了树结构（或可以画成树），在每个节点上要做决策，子树的最优解组合成父节点的最优解。

## ▶ 核心技巧

### 树形 DP 模板（DFS + 后序遍历）

```cpp
vector<int> G[MAXN];
int dp[MAXN][2];

void dfs(int u, int parent) {
    dp[u][0] = 0;  // 不选 u
    dp[u][1] = 1;  // 选 u
    for (int v : G[u]) {
        if (v == parent) continue;
        dfs(v, u);  // 先递归处理子树
        // 后序遍历：用子树结果更新当前节点
        dp[u][0] += max(dp[v][0], dp[v][1]);
        dp[u][1] += dp[v][0];
    }
}
```

**关键顺序：** 先递归处理子节点，再在父节点汇总——这是"后序遍历"。

### 树的直径（DP 做法）

```cpp
int diameter = 0;

int dfs(int u, int parent) {
    int max1 = 0, max2 = 0;
    for (int v : G[u]) {
        if (v == parent) continue;
        int d = dfs(v, u) + 1;
        if (d > max1) { max2 = max1; max1 = d; }
        else if (d > max2) max2 = d;
    }
    diameter = max(diameter, max1 + max2);
    return max1;  // 返回从 u 向下的最长链
}
```

## ▶ 真题 1：城市规划

**试题编号：201909-5（第17次认证第5题）**

n 个城市的树形路网，要选 m 个城市建设公共设施。每个设施有建设成本。居民去最近的设施享受服务。求最小的总成本 + 总距离之和。

**思路：** 树形 DP。dp[u][k] 表示在子树 u 中建设 k 个设施的最小代价。合并子树时做**分组背包**：每棵子树 v 提供的"k 个设施"是一组内只能选一个的物品。

## ▶ 真题 2：食材运输（综合）

**试题编号：202012-4（第19次认证第4题）**

n 个节点的树，k 种食材，m 辆运输车。求满足所有需求的最小时间。

**思路：** 二分答案 + 树形 DP + 状压 DP：
1. 二分最大运输时间 T
2. 树形 DP 计算在时间 T 内，一个起点能覆盖哪些食材种类（用 mask 表示）
3. 状压 DP 选 m 个起点，使它们覆盖全部 k 种食材

**CSP 考察点：** 这是 CSP T4-T5 的满分做法——多算法组合。每个模块单独看都不太难，组合起来需要清晰的设计。

---

## ▶ 树形 DP 常见技巧

- **换根 DP：** 第一次 DFS 算子树信息，第二次 DFS 从根往下传父节点信息
- **分组背包合并子树：** 每个子节点是一组，只能选一种"分配给子树的资源量"
- **二次扫描：** 常用于求"所有节点到某点的距离之和"
