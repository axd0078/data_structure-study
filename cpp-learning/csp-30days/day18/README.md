# Day 18：二分查找与二分答案

## ▶ CSP 考情分析

二分法在 CSP 中是最重要的优化工具之一，**第 2-4 题都可能出现**。CSP 考二分有两个层次：
1. **二分查找：** 在有序序列中找某个值（基础）
2. **二分答案：** 把一个"求最优值"的问题转化为"判断某个值是否可行"（进阶）

**CSP 二分答案的信号：**
- "最大值最小 / 最小值最大"
- "能否在 X 条件下完成"
- 答案具有单调性（如果 k 可行，那么 k+1 也可行）

| 真题 | 二分类型 |
|------|---------|
| 解密(2022-J T2) | 二分查找精确解 |
| 校门外的树(202104-4) | 二分答案 + 验证 |
| 食材运输(202012-4) | 二分答案 + 树形 DP |

## ▶ 核心技巧

### 二分查找模板

```cpp
int binary_search(int a[], int n, int target) {
    int l = 0, r = n - 1;
    while (l <= r) {
        int mid = l + (r - l) / 2;  // 防溢出写法
        if (a[mid] == target) return mid;
        else if (a[mid] < target) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}
```

### 二分答案模板

```cpp
// 求"最大值最小" → pattern
while (l < r) {
    int mid = l + (r - l) / 2;
    if (check(mid))   // mid 可行
        r = mid;      // 尝试更小
    else
        l = mid + 1;  // 必须更大
}
return l;
```

**关键：** `check(mid)` 函数是二分答案的核心——判断"答案 = mid 时能否满足条件"。

### upper_bound / lower_bound（STL）

```cpp
int pos = lower_bound(a, a + n, x) - a;   // 第一个 ≥ x 的位置
int pos = upper_bound(a, a + n, x) - a;   // 第一个 > x 的位置
```

## ▶ 真题 1：解密

**来源：2022 CSP-J 第 2 题**

给定 n, e, d，求两个正整数 p, q 满足 p×q=n 且 (p-1)(q-1)+1 = e×d。

**思路：** m = n - e×d + 2，则 p+q = m，p×q = n。二分 p ∈ [1, m/2]，使 p×(m-p) = n。

## ▶ 真题 2：校门外的树

**试题编号：202104-4（第22次认证第4题）**

n 个障碍物，在每个"空隙"中可以种树，树的间距有若干种选择。求在 [1, m] 范围内有多少种方案。

**思路：** 对间距 k 用二分（或直接枚举约数），对每个 k 判断"所有障碍物之间都能种下间距为 k 的树"。

**CSP 考察点：** 二分答案 + 约数预处理——"判断可行性"比"直接求方案数"更简单。

---

## ▶ 二分在 CSP 中的进阶应用

**食材运输 (202012-4)：** 二分总时间上限，对给定的时间上限，用树形 DP + 状压 DP 判断是否能在该时间内完成所有配送。这是 CSP T4-T5 的经典模式：二分答案减少一维搜索空间。
