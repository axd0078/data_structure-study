# Day 2：排序与相邻关系

## ▶ CSP 考情分析

CSP 中排序一般不单独成题，但**排序后的"相邻性质"是第 1-2 题的高频考点**。

**CSP 为什么爱考排序后相邻？** 因为排序把"任意两数之间的关系"变成了"相邻两数之间的关系"——复杂度从 O(n²) 降到 O(n log n)。

| 问题 | 真题 | 排序后怎么做 |
|------|------|------------|
| 找相差 1 的数对 | 相邻数对(201409-1) | 相邻判断 `a[i]-a[i-1]==1` |
| 找差值最小的两个数 | 最小差值(201712-1) | 相邻差取 min |
| 按频次排序输出 | 数字排序(201503-2) | 结构体排序 + 自定义 cmp |

## ▶ 核心技巧

### sort 的三种用法

```cpp
#include <algorithm>

// 1. 数组
int a[1000];
sort(a, a + n);

// 2. vector
vector<int> v;
sort(v.begin(), v.end());

// 3. 自定义比较器（CSP 第 2 题常用）
struct Node { int val, cnt; };
bool cmp(const Node& a, const Node& b) {
    if (a.cnt != b.cnt) return a.cnt > b.cnt;  // 频次高的在前
    return a.val < b.val;                        // 频次相同按值升序
}
sort(v.begin(), v.end(), cmp);
```

**CSP 排序题的坑：** 第 2 题如 数字排序(201503-2) 需要"按频次降序，频次相同按值升序"——这种二维排序是 CSP 最爱的写法考察。

## ▶ 真题 1：相邻数对

**试题编号：201409-1（第2次认证第1题）**

给定 n 个不同整数，求相差 1 的数对个数。

| 输入样例 | 输出样例 |
|---------|---------|
| 6\n10 2 6 3 7 8 | 3 |

差 1 的数对：(2,3), (6,7), (7,8)

**思路：** 排序 → 遍历相邻，`a[i] - a[i-1] == 1` 则计数 +1

---

## ▶ 真题 2：最小差值

**试题编号：201712-1（第7次认证第1题）**

n 个数（不保证互异），求差值绝对值最小的两个数的差值。

| 输入样例 | 输出样例 |
|---------|---------|
| 5\n1 5 4 8 20 | 1 |
| 5\n9 3 6 1 3 | 0 |

**与相邻数对的区别：** 这个题允许重复值，排序后相邻差可能为 0。

**思路：** 排序 → 遍历相邻，`ans = min(ans, a[i] - a[i-1])`

## ▶ 真题 3：数字排序（进阶级）

**试题编号：201503-2（第4次认证第2题）**

n 个整数，按出现频次从高到低输出，频次相同按值从小到大。

| 输入样例 | 输出样例 |
|---------|---------|
| 12\n5 2 3 3 1 3 4 2 5 2 3 5 | 3 4\n2 3\n5 3\n1 1\n4 1 |

**CSP 考察点：** 这是从"排序解决相邻关系"升级到"用排序做统计输出"——需要先统计频次，再对统计结果排序。用 `map` + 自定义比较器。
