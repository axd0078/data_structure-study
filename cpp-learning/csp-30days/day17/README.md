# Day 17：双指针与滑动窗口

## ▶ CSP 考情分析

双指针/滑动窗口在 CSP 中是**第 2 题的常见优化手段**。很多题暴力的做法是 O(n²) 双循环，但用双指针可以降到 O(n)。

**CSP 双指针题的典型信号：**
- "找连续子数组/子串满足某条件"
- "对每个位置，找最远能延伸到哪里"
- "两个有序数组/序列的处理"

| 真题 | 双指针类型 |
|------|-----------|
| 出行计划(202203-2) | 扫描线 + 双指针维护窗口 |
| 邻域均值(202104-2) | 滑动窗口求区域和 |
| 仓库规划(202309-1) | 多维比较，双指针优化 |
| 序列查询新解(202112-2) | 分段统计 + 双指针 |

## ▶ 核心技巧

### 同向双指针（滑动窗口）

```cpp
int l = 0;
for (int r = 0; r < n; r++) {
    add(a[r]);                    // 右指针扩张
    while (condition() == false) {
        remove(a[l]);             // 左指针收缩
        l++;
    }
    ans = max(ans, r - l + 1);   // 更新答案
}
```

### 相向双指针

```cpp
int l = 0, r = n - 1;
while (l < r) {
    if (a[l] + a[r] == target) { /* 找到 */ }
    else if (a[l] + a[r] < target) l++;
    else r--;
}
```

### 快慢指针

```cpp
int slow = 0;
for (int fast = 0; fast < n; fast++) {
    if (shouldKeep(a[fast])) {
        a[slow++] = a[fast];   // 保留的元素移到前面
    }
}
```

## ▶ 真题 1：邻域均值

**试题编号：202104-2（第22次认证第2题）**

n×n 灰度图像，对每个像素计算其邻域（大小为 r 的矩形）内所有像素的灰度均值和方差，判断是否为"较暗区域"。

**思路：** 二维前缀和维护邻域灰度之和，O(1) 求每个像素邻域的均值和方差。

**为什么用前缀和而非滑动窗口？** 二维情况下前缀和更方便，但思想类似——预计算避免重复扫描。

## ▶ 真题 2：仓库规划

**试题编号：202309-1（第32次认证第1题）**

有 n 个 m 维仓库，对每个仓库找编号最小的"上级仓库"（上级要求在所有维度上都严格大于）。

| 输入样例 | 输出样例 |
|---------|---------|
| 4 2\n0 0\n-1 -1\n1 2\n0 -1 | 3\n1\n0\n3 |

**思路：** 对每个仓库 i，遍历所有仓库 j，检查 j 在所有维度上是否严格大于 i，取编号最小的。

**优化方向：** 如果 n 很大，可以按第一维排序后用双指针缩小搜索范围。CSP 第 1 题通常 n 不大，暴力可过。

## ▶ 真题 3：序列查询新解

**试题编号：202112-2（第24次认证第2题）**

给定递增序列 A 和 N。定义 f(x) 和 g(x)，求 Σ|f(x) - g(x)| for x=0..N-1。

**思路：** f(x) 分段常数——A[i] ≤ x < A[i+1] 时 f(x)=i。g(x) 也是分段常数。用双指针同时扫描 f 和 g 的分段，处理每个共同段的贡献。

**CSP 考察点：** 这是一道"分段扫描+双指针"的题，如果暴力枚举每个 x 会超时。关键是意识到 f 和 g 都是分段常数函数。
