# Day 035：图搜索与阈值区间合并

## 学习内容

今天的两题一个是小图可达，一个是阈值反推。

机器人复健指南里，机器人每步走“日”字，也就是国际象棋中马的 8 个方向。`n,k<=100`，直接从起点做 BFS，记录每个格子最早被访问的步数，超过 `k` 步就不继续扩展。输出访问过的格子数量。

水印检查不要枚举每个阈值再扫整张图。对每个 `5 x 9` 子矩阵，可以直接反推出使它显现出 `CSP` 的阈值区间：所有白色位置都要 `>=k`，所有黑色位置都要 `<k`。因此可行 `k` 满足：

```text
max_black + 1 <= k <= min_white
```

每个匹配窗口贡献一个阈值区间，用差分数组合并所有窗口，最后输出被至少一个窗口覆盖的阈值。

## 题解

### 第38次 T2 机器人复健指南

**题意抓手**

在 `n x n` 棋盘中，机器人从 `(x,y)` 出发，每一步可以跳到 8 个“日”字方向之一。求不超过 `k` 步可以到达多少个不同格子，起点也算可达。

**知识点**

网格 BFS。边权都是 1，且只关心 `k` 步内可达的点，用队列保存 `(x,y,dist)`。第一次访问某格子时就是最短步数，之后无需重复入队。

**解法**

准备 8 个方向数组。起点标记已访问并入队。每次弹出队首，如果 `dist==k` 就不再扩展；否则枚举 8 个目标位置，若在棋盘内且未访问，则标记、计数并入队。

**易错点**

- 坐标从 1 到 `n`，不是 0 到 `n-1`。
- 起点本身要计入答案。
- “不超过 k 步”意味着小于等于 `k` 的所有层都算。

**复杂度**

棋盘最多 `100 x 100`，每个格子最多入队一次，时间复杂度 `O(n^2)`，空间复杂度 `O(n^2)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    int sx, sy;
    cin >> n >> k >> sx >> sy;

    vector<pair<int, int>> moves = {
        {1, 2}, {2, 1}, {-1, 2}, {-2, 1},
        {1, -2}, {2, -1}, {-1, -2}, {-2, -1},
    };

    vector<vector<int>> seen(n + 1, vector<int>(n + 1, 0));
    queue<tuple<int, int, int>> q;
    seen[sx][sy] = 1;
    q.push({sx, sy, 0});

    int ans = 1;
    while (!q.empty()) {
        auto [x, y, dist] = q.front();
        q.pop();
        if (dist == k) {
            continue;
        }
        for (auto [dx, dy] : moves) {
            int nx = x + dx;
            int ny = y + dy;
            if (1 <= nx && nx <= n && 1 <= ny && ny <= n && !seen[nx][ny]) {
                seen[nx][ny] = 1;
                ++ans;
                q.push({nx, ny, dist + 1});
            }
        }
    }

    cout << ans << '\n';
    return 0;
}
```

### 第39次 T2 水印检查

**题意抓手**

给定 `n x n` 灰度图和灰度范围 `[0,L-1]`。选定阈值 `k` 后，像素值 `>=k` 为白色，否则为黑色。要求输出所有能在图中检测出固定 `5 x 9` 的 `CSP` 水印形状的阈值 `k`。

**知识点**

阈值区间反推。对一个窗口来说，水印白色位置要求 `k <= 这些位置的最小灰度`，黑色位置要求 `k > 这些位置的最大灰度`。所以窗口对应的可行区间是 `[max_black+1, min_white]`。

**解法**

枚举所有 `5 x 9` 子矩阵。对每个窗口，按固定水印模板统计 `min_white` 和 `max_black`。若 `max_black+1 <= min_white`，说明存在可行阈值，用差分数组给这个区间加 1。最后扫描 `k=0..L-1` 的差分前缀，前缀值大于 0 的阈值就是至少被一个窗口支持的答案。

**易错点**

- 白色条件是 `>=k`，黑色条件是 `<k`，所以黑色给出的下界是 `max_black+1`。
- 若多个窗口支持同一阈值，只输出一次。
- 差分数组需要开到 `L+1`，因为会在 `high+1` 处减。

**复杂度**

窗口数为 `(n-4)(n-8)`，每个窗口检查固定 45 个格子，时间复杂度 `O(n^2)`；差分扫描 `O(L)`，空间复杂度 `O(n^2+L)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

const vector<string> WATERMARK = {
    "*********",
    "*--*--*-*",
    "*--*****-",
    "*----**--",
    "*******--",
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, limit;
    cin >> n >> limit;
    vector<vector<int>> a(n, vector<int>(n));
    for (auto& row : a) {
        for (int& value : row) {
            cin >> value;
        }
    }

    vector<int> diff(limit + 1, 0);
    for (int top = 0; top + 5 <= n; ++top) {
        for (int left = 0; left + 9 <= n; ++left) {
            int min_white = limit - 1;
            int max_black = -1;
            for (int i = 0; i < 5; ++i) {
                for (int j = 0; j < 9; ++j) {
                    int value = a[top + i][left + j];
                    if (WATERMARK[i][j] == '*') {
                        min_white = min(min_white, value);
                    } else {
                        max_black = max(max_black, value);
                    }
                }
            }

            int low = max_black + 1;
            int high = min_white;
            if (low <= high) {
                ++diff[low];
                --diff[high + 1];
            }
        }
    }

    int active = 0;
    for (int k = 0; k < limit; ++k) {
        active += diff[k];
        if (active > 0) {
            cout << k << '\n';
        }
    }

    return 0;
}
```
