Day 009 的两道题都在练“把定义直接翻译成线性统计”。不要为了看起来通用而排序或枚举所有可能数组，题面已经给出了可以直接利用的结构。

- **灰度直方图**：灰度值范围是 `[0,L)`，天然适合计数桶。
- **数组推导**：`B_i` 是 `A` 的前缀最大值。`sum(A)` 最大时每个 `A_i` 都取 `B_i`；最小时，只有 `B_i` 比前一个前缀最大值变大时，`A_i` 必须取 `B_i`，否则可以取 0。

复杂度：

- 灰度直方图：`O(nm + L)` 时间，`O(L)` 空间。
- 数组推导：`O(n)` 时间，`O(1)` 空间。

## 题解

### 第22次 T1 灰度直方图

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

给出一个 `n*m` 的灰度矩阵，每个像素值在 `[0,L)`。要求输出长度为 `L` 的直方图，`h[x]` 表示灰度值 `x` 出现了多少次。

**知识点**

值域是连续整数 `[0,L)`，所以直接开 `counts[L]`。读到一个像素 `value`，执行 `counts[value]++`。

矩阵结构对答案没有额外影响，按 `n*m` 个数顺序读入即可。

**解法**

1. 读入 `n,m,L`。
2. 初始化 `counts` 长度为 `L`。
3. 读入 `n*m` 个像素并计数。
4. 输出 `counts[0]` 到 `counts[L-1]`，空格分隔。

**易错点**

- 灰度范围是 `0` 到 `L-1`，不是 `1` 到 `L`。
- 输出只有一行，包含 `L` 个数。
- 不要漏掉出现次数为 0 的灰度值。

**复杂度**

时间复杂度 `O(nm + L)`，额外空间复杂度 `O(L)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m, limit;
    cin >> n >> m >> limit;
    vector<int> counts(limit);
    for (int i = 0; i < n * m; i++) {
        int value;
        cin >> value;
        ++counts[value];
    }
    for (int i = 0; i < limit; i++) {
        if (i) {
            cout << ' ';
        }
        cout << counts[i];
    }
    cout << '\n';
    return 0;
}
```

### 第23次 T1 数组推导

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

数组 `B` 是数组 `A` 的前缀最大值：

```text
B_i = max(A_1, ..., A_i)
```

已知非降的 `B`，求所有可能的 `A` 中，`sum(A)` 的最大值和最小值。

**知识点**

最大值很直接：因为 `A_i <= B_i`，且取 `A_i = B_i` 一定合法，所以最大和就是 `sum(B)`。

最小值看前缀最大值何时变大：

- 如果 `B_i > B_{i-1}`，说明第 `i` 个位置必须出现这个新的最大值，因此 `A_i = B_i`。
- 如果 `B_i == B_{i-1}`，说明前面已经有这个最大值了，当前 `A_i` 可以取 0。

`B_1` 一定要计入最小和。

**解法**

1. 扫描 `B`。
2. 最大和累加所有 `B_i`。
3. 用 `previous` 记录上一个 `B`。
4. 如果当前 `B_i > previous`，最小和加上 `B_i`。
5. 输出最大和和最小和。

**易错点**

- `B_i` 可以为 0，所以 `previous` 初始值要小于 0。
- 最大和不是最后一个值乘 `n`，而是所有 `B_i` 的和。
- 最小和只在前缀最大值上升时加。

**复杂度**

时间复杂度 `O(n)`，额外空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    long long maxSum = 0;
    long long minSum = 0;
    int previous = -1;
    for (int i = 0; i < n; i++) {
        int value;
        cin >> value;
        maxSum += value;
        if (value > previous) {
            minSum += value;
        }
        previous = value;
    }
    cout << maxSum << '\n' << minSum << '\n';
    return 0;
}
```
