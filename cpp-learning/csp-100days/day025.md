# Day 025：记录流、集合判邻与稀疏双指针

## 学习内容

今天的三题都在训练“只保存必要信息”。

小明种苹果（续）是一道记录流题。每棵树先给初始苹果数，后续记录可能是疏果操作，也可能是重新统计的苹果数。遇到负数就累加减少；遇到正数表示重新统计，如果它小于当前估计值，则说明发生掉落。最后输出剩余总数、发生掉落的树数、连续三棵都掉落的组数。

回收站选址是二维点集合判邻。一个点能作为回收站，必须上下左右四个相邻点都存在；评分则看四个对角点存在几个。坐标范围大，不能开二维数组，应该用 `set<pair<int,int>>` 或 `unordered_set` 存点并查询。

稀疏向量训练双指针合并。两个向量只给非零项，并且下标递增。点积只在两个向量下标相同时产生贡献，所以用两个指针扫两个非零项列表，谁的下标小谁前进。

## 题解

### 第17次 T2 小明种苹果（续）

**题意抓手**

有 `n` 棵苹果树。每棵树的一行记录先给操作次数和初始苹果数，后面每个数如果小于等于 0 表示疏果数量，直接加到当前苹果数；如果大于 0 表示重新统计出的苹果数。若重新统计值小于当前估计值，则该树发生过苹果掉落。

**知识点**

记录流模拟和环形邻接统计。每棵树只需要维护当前估计值和是否掉落。最后连续三棵树要按环形判断，`i` 的左右邻居是 `(i-1+n)%n` 和 `(i+1)%n`。

**解法**

逐棵树读取。`current` 初始化为该树第一条苹果数。后续记录中，负数或 0 累加到 `current`；正数表示重新计数，如果 `value < current`，标记掉落，然后令 `current=value`。累加所有树最终 `current` 得到 `T`，统计掉落树数得到 `D`，再枚举每棵树是否和左右邻居都掉落得到 `E`。

**易错点**

- 正数不是增加苹果，而是一次重新统计。
- 发生掉落后仍要用正数记录覆盖当前值。
- 三连统计是环形，第 1 棵和第 `n` 棵相邻。

**复杂度**

每条记录只处理一次，时间复杂度 `O(记录总数 + n)`，空间复杂度 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    long long total = 0;
    vector<int> dropped(n, 0);

    for (int i = 0; i < n; ++i) {
        int m;
        cin >> m;
        long long current;
        cin >> current;

        for (int j = 1; j < m; ++j) {
            long long value;
            cin >> value;
            if (value <= 0) {
                current += value;
            } else {
                if (value < current) dropped[i] = 1;
                current = value;
            }
        }

        total += current;
    }

    int d = accumulate(dropped.begin(), dropped.end(), 0);
    int e = 0;
    for (int i = 0; i < n; ++i) {
        if (dropped[(i - 1 + n) % n] && dropped[i] && dropped[(i + 1) % n]) {
            ++e;
        }
    }

    cout << total << ' ' << d << ' ' << e << '\n';
    return 0;
}
```

### 第18次 T2 回收站选址

**题意抓手**

给定若干垃圾点坐标。一个点能作为回收站，要求它上下左右四个相邻位置都有垃圾。若能作为回收站，它的评分等于四个对角相邻位置中有垃圾的个数。输出评分 0 到 4 的候选点数量。

**知识点**

集合判存在。坐标范围不能直接开数组，用 `set<pair<int,int>>` 存所有点。方向数组让上下左右和四个对角的检查更统一。

**解法**

枚举每个垃圾点 `(x,y)`。先检查 `(x+1,y)`、`(x-1,y)`、`(x,y+1)`、`(x,y-1)` 是否都在集合中；如果不是，跳过。否则统计四个对角点存在数量 `score`，令 `ans[score]++`。最后逐行输出 `ans[0..4]`。

**易错点**

- 只有垃圾点本身才能作为候选回收站。
- 上下左右是必要条件，对角只用于评分。
- 输出 5 行，不是一行。

**复杂度**

每个点做常数次集合查询，`set` 查询为 `O(log n)`，总时间复杂度 `O(n log n)`，空间复杂度 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    set<pair<int, int>> points;
    for (int i = 0; i < n; ++i) {
        int x, y;
        cin >> x >> y;
        points.insert({x, y});
    }

    vector<int> ans(5, 0);
    int dx4[4] = {1, -1, 0, 0};
    int dy4[4] = {0, 0, 1, -1};
    int dxc[4] = {1, 1, -1, -1};
    int dyc[4] = {1, -1, 1, -1};

    for (auto [x, y] : points) {
        bool ok = true;
        for (int i = 0; i < 4; ++i) {
            ok &= points.count({x + dx4[i], y + dy4[i]});
        }
        if (!ok) continue;

        int score = 0;
        for (int i = 0; i < 4; ++i) {
            score += points.count({x + dxc[i], y + dyc[i]});
        }
        ++ans[score];
    }

    for (int value : ans) {
        cout << value << '\n';
    }

    return 0;
}
```

### 第19次 T2 稀疏向量

**题意抓手**

两个 `n` 维向量只给出非零项。第一个向量有 `a` 个非零项，第二个向量有 `b` 个非零项，每项为下标和值。要求两个向量的点积。

**知识点**

有序稀疏表合并。非零项按下标递增给出，只有两个指针指向相同下标时才贡献乘积。答案可能超过 32 位，要用 `long long`。

**解法**

读入两个 `(index,value)` 列表。令 `i=0,j=0`。如果 `u[i].index == v[j].index`，累加乘积并同时前进；如果前者下标小，前进 `i`；否则前进 `j`。

**易错点**

- 维度 `n` 很大，但不需要开长度为 `n` 的数组。
- 乘积和答案要用 64 位。
- 只在下标相同的位置相乘。

**复杂度**

两个非零项列表各扫描一次，时间复杂度 `O(a+b)`，空间复杂度 `O(a+b)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long n;
    int a, b;
    cin >> n >> a >> b;
    vector<pair<int, long long>> u(a), v(b);
    for (auto& [idx, value] : u) cin >> idx >> value;
    for (auto& [idx, value] : v) cin >> idx >> value;

    int i = 0, j = 0;
    long long ans = 0;
    while (i < a && j < b) {
        if (u[i].first == v[j].first) {
            ans += u[i].second * v[j].second;
            ++i;
            ++j;
        } else if (u[i].first < v[j].first) {
            ++i;
        } else {
            ++j;
        }
    }

    cout << ans << '\n';
    return 0;
}
```
