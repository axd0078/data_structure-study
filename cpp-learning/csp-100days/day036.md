# Day 036：有限状态逆变换与混合背包

## 学习内容

今天两题都不是靠套模板，而是先把题意转成更小、更稳定的模型。

《数字变换》表面上有 `n<=5*10^5, m<=1000`，不能对每个查询从 `0..511` 暴力试一遍。但值域只有 `2^9=512` 个状态，每次 `g(x,k)` 都是这 512 个状态上的置换。做法有两种：一种是先把完整变换 `F` 对 512 个输入的输出全部算出来，再建反表；另一种是直接推 `g` 的逆变换，然后对每个输出倒着套 `k_m..k_1`。本题参考代码采用逆变换，整体复杂度是 `O(512*m+n)`。

《机器人项目管理》是“普通型 0/1 选择 + 灵活型按单位收益贪心”的混合模型。普通型任务要么花 `a_i` 杯咖啡获得 `b_i` 的缩短，要么不选，所以是 0/1 背包；灵活型任务每杯咖啡的收益固定为 `b_i/a_i`，最多取 `a_i` 杯，把所有灵活型咖啡单位按收益从大到小取即可。最后枚举普通型用了多少咖啡，和剩余咖啡能获得的灵活型收益合并。

## 题解

### 第40次 T2 数字变换 (transform)

**题意抓手**

一个 9 位二进制数被分成高、中、低三段，每段 3 位。每次变换由参数 `k` 决定，给出变换后的 `a_i`，要求反推出原始输入。题目保证答案唯一。

**知识点**

状态压缩和置换逆推。因为合法值只有 `0..511`，不需要在大输入上做复杂搜索。关键是把一次 `g` 的逆变换写清楚。

设输入三段为 `(a,b,c)`，一次正向变换输出为：

```text
(b, c xor f(b,k), a xor f(c,k))
```

如果已知输出三段 `(B,M,L)`，则：

```text
b = B
c = M xor f(b,k)
a = L xor f(c,k)
```

这就能从输出唯一恢复输入。

**解法**

对每个可能输出 `0..511`，倒序应用所有参数 `k_m..k_1` 的逆变换，得到它对应的原始输入，存入 `decoded[output]`。之后每个查询 `a_i` 直接输出 `decoded[a_i]`。

**易错点**

- 三段的顺序是高三位、中三位、低三位，不能按十进制位理解。
- `f(x,k)` 里模 `2^3` 可以写成 `& 7`，但前提是 `x,k` 都非负。
- 查询数很大，输出时注意空格格式。

**复杂度**

预处理 `512` 个状态、每个状态倒推 `m` 次，时间复杂度 `O(512*m+n)`，空间复杂度 `O(512)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int transform_value(int x, int key) {
    auto f = [&](int value) {
        return ((value * value + key * key) & 7) ^ key;
    };

    int b = x >> 6;
    int middle = (x >> 3) & 7;
    int low = x & 7;
    int c = middle ^ f(b);
    int a = low ^ f(c);
    return (a << 6) | (b << 3) | c;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<int> keys(m);
    for (int& key : keys) {
        cin >> key;
    }

    vector<int> decoded(512);
    for (int value = 0; value < 512; ++value) {
        int x = value;
        for (int i = m - 1; i >= 0; --i) {
            x = transform_value(x, keys[i]);
        }
        decoded[value] = x;
    }

    for (int i = 0; i < n; ++i) {
        int value;
        cin >> value;
        if (i) {
            cout << ' ';
        }
        cout << decoded[value];
    }
    cout << '\n';
    return 0;
}
```

### 第41次 T2 机器人项目管理（crashing）

**题意抓手**

有 `m` 杯咖啡，给机器人分配后减少项目总耗时。普通型任务必须一次性分配 `a_i` 杯才获得 `b_i` 的缩短；灵活型任务可以分配 `0..a_i` 杯，每杯收益是 `b_i/a_i`。求最短总时间。

**知识点**

普通型是 0/1 背包，灵活型是按单位收益排序的贪心。两个部分用“枚举普通型占用容量”合并。

**解法**

先把所有任务耗时求和为 `total_time`。普通型任务做一维 0/1 背包，`dp[j]` 表示用 `j` 杯咖啡在普通型任务中最多缩短多少时间。灵活型任务按 `b_i/a_i` 从大到小排序，逐杯累加得到 `flexible_saved[c]`，表示给灵活型最多 `c` 杯咖啡能获得的最大缩短。最后枚举普通型使用 `used` 杯，答案取 `dp[used]+flexible_saved[m-used]` 最大值。

**易错点**

- 普通型不能按单位收益拆开，否则会把“必须一次性投入”的条件做错。
- 灵活型可以拆杯，排序比较用交叉乘法或 double 都可以；参考代码只用于排序，误差不会影响 `0.001` 的输出要求。
- 输出实数，建议固定 6 位小数。

**复杂度**

普通型背包 `O(nm)`，灵活型排序和预处理 `O(A log A + m)`，其中 `A` 是灵活型任务数；由于只按任务排序，实际为 `O(n log n + nm + m)`。空间复杂度 `O(m)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Task {
    int coffee;
    double saved;
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<Task> normal, flexible;
    double total_time = 0.0;

    for (int i = 0; i < n; ++i) {
        int type, t, a, b;
        cin >> type >> t >> a >> b;
        total_time += t;
        if (type == 0) {
            flexible.push_back({a, double(b)});
        } else {
            normal.push_back({a, double(b)});
        }
    }

    vector<double> dp(m + 1, 0.0);
    for (const Task& task : normal) {
        for (int used = m; used >= task.coffee; --used) {
            dp[used] = max(dp[used], dp[used - task.coffee] + task.saved);
        }
    }

    sort(flexible.begin(), flexible.end(), [](const Task& lhs, const Task& rhs) {
        return lhs.saved * rhs.coffee > rhs.saved * lhs.coffee;
    });

    vector<double> flexible_saved(m + 1, 0.0);
    int capacity = 0;
    double saved = 0.0;
    for (const Task& task : flexible) {
        for (int take = 1; take <= task.coffee && capacity < m; ++take) {
            ++capacity;
            saved += task.saved / task.coffee;
            flexible_saved[capacity] = saved;
        }
        if (capacity == m) {
            break;
        }
    }
    for (int i = capacity + 1; i <= m; ++i) {
        flexible_saved[i] = saved;
    }

    double best_saved = 0.0;
    for (int used = 0; used <= m; ++used) {
        best_saved = max(best_saved, dp[used] + flexible_saved[m - used]);
    }

    cout << fixed << setprecision(6) << total_time - best_saved << '\n';
    return 0;
}
```
