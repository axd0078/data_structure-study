# Day 015：行优先顺序与规则判定

## 学习内容

Day 015 的两道题都很适合练“按定义实现，但不要多做事”。矩阵重塑不需要真的理解矩阵变换，只要抓住行优先展开后元素顺序不变；密码判定不需要复杂分类器，只要按题面规则维护几个布尔状态和字符计数。

**矩阵重塑（其一）**的核心是行优先顺序。原矩阵按行读入，重塑后的矩阵也按同一条一维序列按行输出。因此可以一边读入元素一边输出，只根据目标列数 `q` 决定什么时候换行，不需要保存整个矩阵。

**密码**要判断三个等级。题目保证每个输入至少满足低安全要求，所以只需要判断是否达到中、高：

- 必须同时包含字母、数字、特殊字符 `*` 或 `#`，否则为低 `0`。
- 如果满足上条但某个字符出现超过 2 次，为中 `1`。
- 如果满足上条且所有字符出现次数都不超过 2，为高 `2`。

今天的最低复杂度目标：

- 矩阵重塑（其一）：`O(nm)` 时间，`O(1)` 额外空间，流式输出。
- 密码：`O(total_length)` 时间，`O(1)` 空间，字符种类固定。

## 题解

### 第34次 T1 矩阵重塑（其一）

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

输入原矩阵规模 `n x m`、目标矩阵规模 `p x q`，并保证 `n*m = p*q`。接着输入原矩阵的所有元素，要求按行优先顺序重塑成 `p x q` 并输出。

**知识点**

行优先展开时，原矩阵位置 `(i,j)` 对应一维下标：

```text
k = i * m + j
```

重塑后，一维下标 `k` 对应：

```text
new_row = k / q
new_col = k % q
```

但本题只需要输出完整矩阵，不需要查询单个位置。既然输入顺序已经是原矩阵的行优先顺序，输出也要按目标矩阵的行优先顺序，所以可以读到第 `k` 个元素就直接输出到目标位置。

**解法**

1. 读入 `n,m,p,q`。
2. 总元素数 `total = n*m`。
3. 对 `index=0..total-1`：
   - 读入当前元素。
   - 如果 `index % q != 0`，先输出一个空格。
   - 输出当前元素。
   - 如果 `index % q == q-1`，输出换行。

**易错点**

- 输入和输出都是行优先，不要按列输出。
- 每行数字之间有空格，行尾不需要多余空格。
- 不需要保存矩阵，保存反而让空间复杂度变高。
- `n*m` 最大 `10^4`，但流式写法依然是最简洁的。

**复杂度**

时间复杂度 `O(nm)`，额外空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m, p, q;
    cin >> n >> m >> p >> q;

    int total = n * m;
    for (int index = 0; index < total; ++index) {
        int value;
        cin >> value;

        if (index % q) {
            cout << ' ';
        }
        cout << value;
        if (index % q == q - 1) {
            cout << '\n';
        }
    }

    return 0;
}
```

### 第35次 T1 密码

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

输入 `n` 个候选密码。每个密码由大小写字母、数字、`*`、`#` 组成，长度至少满足低安全要求。对每个密码输出安全等级：

```text
2：高
1：中
0：低
```

高安全要求：长度至少 6，包含字母、数字、特殊字符，且同一个字符出现不超过 2 次。

中安全要求：长度至少 6，包含字母、数字、特殊字符，但没有达到高安全要求。

低安全：没有达到中安全要求。题目保证输入至少为低，所以不用处理非法字符。

**知识点**

判断规则可以拆成两层：

1. 是否同时包含三类字符：字母、数字、特殊字符。
2. 是否所有字符出现次数都不超过 2。

只要第一层不满足，答案就是 `0`。第一层满足后，再看第二层，满足输出 `2`，不满足输出 `1`。

**解法**

1. 对每个密码，初始化 `has_letter`、`has_digit`、`has_special`。
2. 用定长数组 `count[256]` 统计每个字符次数。
3. 扫描字符串：
   - 用 `isalpha` 判断字母。
   - 用 `isdigit` 判断数字。
   - 判断是否是 `*` 或 `#`。
   - 字符计数加一。
4. 如果三类字符不全，输出 `0`。
5. 否则检查所有计数是否都不超过 2；是则输出 `2`，否则输出 `1`。

**易错点**

- 大写字母和小写字母都是字母，但字符计数时 `A` 和 `a` 是不同字符。
- 特殊字符只有 `*` 和 `#`。
- 出现次数限制是“同一个字符不超过 2 次”，不是同一类字符不超过 2 次。
- `isalpha`、`isdigit` 建议传入 `unsigned char` 转换后的值，避免字符符号位问题。

**复杂度**

设所有密码总长度为 `L`。时间复杂度 `O(L)`，额外空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    while (n--) {
        string password;
        cin >> password;

        bool has_letter = false;
        bool has_digit = false;
        bool has_special = false;
        array<int, 256> count{};

        for (char raw : password) {
            unsigned char ch = static_cast<unsigned char>(raw);
            if (isalpha(ch)) {
                has_letter = true;
            }
            if (isdigit(ch)) {
                has_digit = true;
            }
            if (raw == '*' || raw == '#') {
                has_special = true;
            }
            ++count[ch];
        }

        if (!(has_letter && has_digit && has_special)) {
            cout << 0 << '\n';
            continue;
        }

        bool high = true;
        for (int amount : count) {
            if (amount > 2) {
                high = false;
                break;
            }
        }

        cout << (high ? 2 : 1) << '\n';
    }

    return 0;
}
```
