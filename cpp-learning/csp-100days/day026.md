# Day 026：连续状态统计与排序阈值

## 学习内容

今天的两题都属于“把判定规则转成可维护状态”。

风险人群筛查的核心是连续命中。对每个人的轨迹逐点判断是否在矩形内，出现过一次命中就是“经过”，最大连续命中长度达到 `k` 就是“逗留”。这种题不要尝试判断两点之间的路径，题面明确只看给出的 `t` 个坐标。

期末预测之最佳阈值是排序扫描。阈值只会取某个已有安全指数 `y_i`。当阈值从小到大移动时，低于阈值的一组同分学生预测结果会从 1 变成 0：真实结果为 0 的学生从错变对，真实结果为 1 的学生从对变错。用这个变化量维护当前正确数即可。

## 题解

### 第20次 T2 风险人群筛查

**题意抓手**

给定高危矩形和 `n` 个人各 `t` 个位置记录。某人任意一个坐标落在矩形内就是经过；连续至少 `k` 个坐标都在矩形内就是逗留。输出经过人数和逗留人数。

**知识点**

矩形包含判断为 `xl <= x <= xr && yd <= y <= yu`。连续命中可以用 `current` 表示当前连续在矩形内的长度，遇到矩形外坐标清零，同时用 `best` 记录最大连续长度。

**解法**

逐人读取轨迹。每读一个点，如果在矩形内，`current++` 并更新 `best`；否则 `current=0`。该人的所有点处理完后，`best>0` 计入经过，`best>=k` 计入逗留。

**易错点**

- 矩形边界也算在内。
- 只看输入的离散坐标，不看两个时刻之间的移动路径。
- “经过”和“逗留”都按人计数，不按坐标点计数。

**复杂度**

时间复杂度 `O(n t)`，空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k, t;
    int xl, yd, xr, yu;
    cin >> n >> k >> t >> xl >> yd >> xr >> yu;

    int passed = 0;
    int stayed = 0;
    for (int i = 0; i < n; ++i) {
        int best = 0;
        int current = 0;
        for (int j = 0; j < t; ++j) {
            int x, y;
            cin >> x >> y;
            if (xl <= x && x <= xr && yd <= y && y <= yu) {
                ++current;
                best = max(best, current);
            } else {
                current = 0;
            }
        }
        passed += best > 0;
        stayed += best >= k;
    }

    cout << passed << '\n' << stayed << '\n';
    return 0;
}
```

### 第21次 T2 期末预测之最佳阈值

**题意抓手**

选择阈值 `theta`，若 `y >= theta` 预测为 1，否则预测为 0。阈值只能从已有的 `y_i` 中选。要求预测正确数最多，若并列取最大的阈值。

**知识点**

排序后按安全指数分组。阈值为当前组的 `y` 时，所有比它小的组预测为 0，其余预测为 1。阈值从一个组移动到下一组时，刚经过的组预测由 1 变 0。

**解法**

初始阈值取最小 `y`，所有人都预测为 1，正确数等于真实结果为 1 的人数。按 `y` 升序枚举每个不同的阈值，先用当前正确数更新答案，再把当前组“移到阈值左边”：其中结果为 0 的人数让正确数增加，结果为 1 的人数让正确数减少。因为阈值递增，使用 `>=` 更新答案即可自然保留更大的并列阈值。

**易错点**

- 相同 `y` 的学生必须作为一组一起跨过，不能一个一个更新并评价。
- 并列正确数取最大的阈值。
- `y_i` 可到 `1e8`，但只排序，不需要开桶。

**复杂度**

排序 `O(m log m)`，扫描 `O(m)`，空间复杂度 `O(m)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int m;
    cin >> m;
    vector<pair<int, int>> a(m);
    int correct = 0;
    for (auto& [y, result] : a) {
        cin >> y >> result;
        correct += result;
    }

    sort(a.begin(), a.end());

    int best = -1;
    int answer = a[0].first;
    for (int i = 0; i < m;) {
        int y = a[i].first;
        if (correct >= best) {
            best = correct;
            answer = y;
        }

        int zeros = 0;
        int ones = 0;
        while (i < m && a[i].first == y) {
            if (a[i].second == 0) ++zeros;
            else ++ones;
            ++i;
        }
        correct += zeros - ones;
    }

    cout << answer << '\n';
    return 0;
}
```
