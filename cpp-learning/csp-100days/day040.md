# Day 040：路径栈与双人对战模拟

## 学习内容

今天是中档模拟复盘，重点是把规则拆成稳定的小函数。

《路径解析》可以看成栈问题。绝对路径从根目录开始，相对路径从当前目录开始；路径段为空或 `.` 时忽略，遇到 `..` 就弹出上一段，但根目录不能继续向上。最终栈为空输出 `/`，否则用 `/` 拼接。

《炉石传说》是双玩家状态模拟。每个玩家有英雄血量和一个随从数组。召唤时在指定位置插入；攻击时双方同时扣血；随从死亡后从数组中删除，右侧编号自动前移。只要把“当前玩家”和“对手”统一成 `current` 与 `current^1`，代码会清晰很多。

## 题解

### 第07次 T3 路径解析

**题意抓手**

给定当前目录和若干路径，把每个路径正规化为绝对路径，去掉连续斜杠、`.`、`..` 和结尾多余斜杠。

**知识点**

栈处理路径段。空字符串也是合法路径，代表当前目录。

**解法**

若待处理路径是绝对路径，从空栈开始；否则先把当前目录拆成路径段压入栈。然后按 `/` 拆待处理路径：空段和 `.` 忽略；`..` 表示返回上一级，栈非空时弹出；普通名称压入。最后根据栈生成结果。

**易错点**

- 输入路径可能是空行，读路径必须用 `getline`。
- 根目录的上一级仍是根目录，栈空时遇到 `..` 不做事。
- 多个连续 `/` 会产生空段，应直接忽略。

**复杂度**

每个路径长度不超过 1000，逐字符处理，时间复杂度 `O(L)`，空间复杂度 `O(L)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<string> split_path(const string& path) {
    vector<string> parts;
    string cur;
    for (char ch : path) {
        if (ch == '/') {
            parts.push_back(cur);
            cur.clear();
        } else {
            cur.push_back(ch);
        }
    }
    parts.push_back(cur);
    return parts;
}

string normalize_path(const string& current, const string& path) {
    vector<string> stack;
    if (path.empty() || path[0] != '/') {
        for (const string& part : split_path(current)) {
            if (!part.empty()) {
                stack.push_back(part);
            }
        }
    }

    for (const string& part : split_path(path)) {
        if (part.empty() || part == ".") {
            continue;
        }
        if (part == "..") {
            if (!stack.empty()) {
                stack.pop_back();
            }
        } else {
            stack.push_back(part);
        }
    }

    if (stack.empty()) {
        return "/";
    }
    string result;
    for (const string& part : stack) {
        result += "/" + part;
    }
    return result;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int p;
    string current;
    cin >> p >> current;
    string path;
    getline(cin, path);
    while (p--) {
        getline(cin, path);
        cout << normalize_path(current, path) << '\n';
    }
    return 0;
}
```

### 第08次 T3 炉石传说

**题意抓手**

模拟两名玩家的英雄和随从。当前玩家可以召唤随从、让随从攻击对方英雄或随从、结束回合。输出最终胜负、英雄血量和双方随从血量。

**知识点**

顺序模拟、动态数组插入删除。随从编号始终是当前数组下标加 1。

**解法**

维护 `Player players[2]`，每个玩家包含英雄血量和随从数组。`current` 表示当前操作方。召唤时在 `position-1` 插入随从；攻击英雄时扣对方英雄血量；攻击随从时先同时扣双方随从血量，再删除死亡随从。`end` 时 `current^=1`。最后根据英雄血量判断胜负。

**易错点**

- 攻击随从时伤害同时发生，不能先删一个再用被删随从的攻击力。
- 删除随从会改变编号，但一次攻击中只涉及攻击前给出的编号。
- 英雄可能为负数，直接按题面输出当前血量。

**复杂度**

操作数 `n<=1000`，随从最多 7 个，数组插入删除都是常数规模。时间复杂度 `O(n)`，空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Minion {
    int attack;
    int health;
};

struct Player {
    int hero = 30;
    vector<Minion> minions;
};

void print_minions(const vector<Minion>& minions) {
    cout << minions.size();
    for (const Minion& minion : minions) {
        cout << ' ' << minion.health;
    }
    cout << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    Player players[2];
    int current = 0;

    while (n--) {
        string action;
        cin >> action;
        if (action == "summon") {
            int position, attack, health;
            cin >> position >> attack >> health;
            auto& board = players[current].minions;
            board.insert(board.begin() + position - 1, {attack, health});
        } else if (action == "attack") {
            int attacker, defender;
            cin >> attacker >> defender;
            Minion& atk = players[current].minions[attacker - 1];
            if (defender == 0) {
                players[current ^ 1].hero -= atk.attack;
            } else {
                Minion& def = players[current ^ 1].minions[defender - 1];
                atk.health -= def.attack;
                def.health -= atk.attack;
                if (def.health <= 0) {
                    players[current ^ 1].minions.erase(players[current ^ 1].minions.begin() + defender - 1);
                }
                if (atk.health <= 0) {
                    players[current].minions.erase(players[current].minions.begin() + attacker - 1);
                }
            }
        } else {
            current ^= 1;
        }
    }

    int result = 0;
    if (players[1].hero <= 0) {
        result = 1;
    } else if (players[0].hero <= 0) {
        result = -1;
    }

    cout << result << '\n';
    cout << players[0].hero << '\n';
    print_minions(players[0].minions);
    cout << players[1].hero << '\n';
    print_minions(players[1].minions);
    return 0;
}
```
