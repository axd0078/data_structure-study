# Day 021：矩阵扫描与状态落点模拟

## 学习内容

今天进入 T2 的连续训练，重点不是“写很长的代码”，而是把题面里的规则稳定地翻译成程序状态。

日期计算训练的是线性扣减模型：把“某年第几天”转成“从 1 月开始逐月扣掉整月天数”。核心基础是闰年判断、月份天数表和循环不变量。循环中始终保持 `day` 表示“还没有归入具体日期的剩余天数”，当 `day <= 当前月天数` 时，当前月和剩余天数就是答案。

消除类游戏训练的是矩阵扫描和标记数组。横向连续段和纵向连续段要同时生效，所以不能边扫描边改原矩阵；必须先用 `erase` 标记所有要清零的位置，再统一输出。扫描连续段时，用双指针 `[j,k)` 表示当前相同数字的一段，这比逐格判断左邻右邻更稳。

俄罗斯方块训练的是小状态模拟。棋盘固定为 `15 x 10`，方块固定为 `4 x 4`，可以把方块中为 1 的格子收集成坐标列表，然后反复判断“整体再向下一行是否还能放”。落点不是靠猜高度，而是用 `can_place(top + 1)` 推到不能继续下落为止。

这三题共同的编码原则：

- 原数据和辅助状态分开，避免扫描过程中污染后续判断。
- 坐标统一使用 0 下标，读入的列号这种 1 下标数据要立刻转换。
- 输出格式先定好：矩阵题每行空格分隔、行末换行；日期题分两行输出。

## 题解

### 第05次 T2 日期计算

**题意抓手**

给定年份 `year` 和这年的第 `day` 天，输出它对应的月和日。第几天从 1 开始计数，所以 `day=1` 表示 1 月 1 日。

**知识点**

闰年规则是基础：能被 400 整除，或能被 4 整除且不能被 100 整除。只需要调整 2 月天数，然后从 1 月起逐月扣减。

**解法**

维护月份天数数组 `days[1..12]`。如果是闰年，令 `days[2]=29`。从 `month=1` 开始，如果 `day > days[month]`，说明目标日期不在当前月，于是 `day -= days[month]`、`month++`。循环停止时，`month` 是月份，`day` 是当月日期。

**易错点**

- `day == days[month]` 时目标就在当前月最后一天，不能继续扣。
- 月份表最好从下标 1 开始，减少 `month-1` 这种偏移。
- 输出是两行：第一行月份，第二行日期。

**复杂度**

时间复杂度 `O(12)`，也就是常数；空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

bool leap(int y) {
    return y % 400 == 0 || (y % 4 == 0 && y % 100 != 0);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int year, day;
    cin >> year >> day;

    int days[] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    if (leap(year)) {
        days[2] = 29;
    }

    int month = 1;
    while (day > days[month]) {
        day -= days[month];
        ++month;
    }

    cout << month << '\n' << day << '\n';
    return 0;
}
```

### 第06次 T2 消除类游戏

**题意抓手**

给定一个 `n x m` 的数字矩阵，横向或纵向连续不少于 3 个相同数字的位置都要变成 0。横向和纵向判断基于原矩阵同时发生。

**知识点**

本题关键是“扫描”和“标记”分离。横向扫描每一行，纵向扫描每一列，发现连续段长度至少为 3 就把这些位置标记为待清除。最后输出时，标记过的位置输出 0，其他位置输出原数。

**解法**

对每一行使用双指针：`j` 是当前连续段起点，`k` 向右扩展到第一个不同数字的位置。如果 `k-j >= 3`，标记这一段。列扫描同理，只是把行列角色换一下。

**易错点**

- 不能扫描到一个连续段就马上改成 0，否则会影响另一方向判断。
- 横向和纵向重叠的位置只需要标记一次。
- 输出时保留原矩阵尺寸和空格格式。

**复杂度**

每个格子在横向扫描和纵向扫描中各被访问常数次，时间复杂度 `O(nm)`，空间复杂度 `O(nm)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<vector<int>> a(n, vector<int>(m));
    vector<vector<int>> erase(n, vector<int>(m, 0));

    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m; ++j) {
            cin >> a[i][j];
        }
    }

    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m;) {
            int k = j;
            while (k < m && a[i][k] == a[i][j]) ++k;
            if (k - j >= 3) {
                for (int t = j; t < k; ++t) erase[i][t] = 1;
            }
            j = k;
        }
    }

    for (int j = 0; j < m; ++j) {
        for (int i = 0; i < n;) {
            int k = i;
            while (k < n && a[k][j] == a[i][j]) ++k;
            if (k - i >= 3) {
                for (int t = i; t < k; ++t) erase[t][j] = 1;
            }
            i = k;
        }
    }

    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m; ++j) {
            if (j) cout << ' ';
            cout << (erase[i][j] ? 0 : a[i][j]);
        }
        cout << '\n';
    }

    return 0;
}
```

### 第07次 T2 俄罗斯方块

**题意抓手**

给定已有棋盘、一个 `4 x 4` 方块形状和方块左边界所在列，模拟方块竖直下落，直到再下落一格会越界或撞到已有方块，然后输出放入后的棋盘。题目不要求消行。

**知识点**

把方块中为 1 的格子抽成坐标列表，可以让合法性检查只遍历 4 个小方块。`can_place(top)` 表示方块左上角在第 `top` 行时是否能放下。

**解法**

读入棋盘和方块，把输入列号转成 0 下标。令 `top=0`，只要 `can_place(top+1)` 为真就继续下落。停止后，把方块的 4 个真实格子写入棋盘并输出。

**易错点**

- 输入列号是 1 下标，要先 `--col`。
- 方块矩阵可能左侧或右侧有空列，不能用矩形边界判断，应该只看值为 1 的格子。
- 题目不要求处理满行消除。

**复杂度**

最多尝试 15 个下落位置，每次检查 4 个格子，时间复杂度 `O(15 * 4)`，空间复杂度 `O(15 * 10)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    vector<vector<int>> board(15, vector<int>(10));
    for (int i = 0; i < 15; ++i) {
        for (int j = 0; j < 10; ++j) {
            cin >> board[i][j];
        }
    }

    vector<pair<int, int>> cells;
    for (int i = 0; i < 4; ++i) {
        for (int j = 0; j < 4; ++j) {
            int x;
            cin >> x;
            if (x == 1) cells.push_back({i, j});
        }
    }

    int col;
    cin >> col;
    --col;

    auto can_place = [&](int top) {
        for (auto [r, c] : cells) {
            int nr = top + r;
            int nc = col + c;
            if (nr < 0 || nr >= 15 || nc < 0 || nc >= 10 || board[nr][nc]) {
                return false;
            }
        }
        return true;
    };

    int top = 0;
    while (can_place(top + 1)) {
        ++top;
    }

    for (auto [r, c] : cells) {
        board[top + r][col + c] = 1;
    }

    for (int i = 0; i < 15; ++i) {
        for (int j = 0; j < 10; ++j) {
            if (j) cout << ' ';
            cout << board[i][j];
        }
        cout << '\n';
    }

    return 0;
}
```
