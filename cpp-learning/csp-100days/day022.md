# Day 022：座位状态、单调函数与顺序维护

## 学习内容

今天的三题分别对应三类常见 T2 模型：固定容量分配、单调函数反推、线性序列移动。

火车购票的车厢只有 20 排、每排 5 个座位，容量固定且很小。优先规则是先找同一排连续空座；如果找不到，再按座位编号从小到大分配零散空座。实现时用 `used[20][5]` 表示占用，座位编号和二维坐标之间用 `row * 5 + col + 1` 转换。

工资计算是反向题。题目给的是税后工资，要求税前工资。直接反推每个税率区间也可以，但更稳的做法是利用“税前工资越高，税后工资也越高”的单调性二分答案。每次写一个 `after_tax(salary)`，二分最小的税前工资，使税后工资不小于目标值。

学生排队训练顺序容器的删除和插入。题目规模不大，`vector` 足够：找到学生当前位置，删除，再按偏移插入。这里真正要稳的是插入位置的含义：题目给的移动量是相对原位置而言，删除后 `vector` 变短，但 `pos + delta` 仍然正好是插入位置。

## 题解

### 第08次 T2 火车购票

**题意抓手**

火车有 100 个座位，分成 20 排、每排 5 个。每次购票人数为 `p`。如果存在某一排连续 `p` 个空座，就分配最靠前的一段；否则按座位编号从小到大分配 `p` 个空座。

**知识点**

固定尺寸状态模拟。用二维数组保存占用情况，再按规则扫描。连续段查找可以枚举每排起点 `start`，检查 `[start,start+p)` 是否都为空。

**解法**

对每个购票请求，先尝试连续分配：从第 1 排到第 20 排，从每排第 1 个可能起点到最后一个可能起点依次检查。找到后立即占座并输出。若没有找到连续座位，再从编号 1 到 100 扫描所有空座，取前 `p` 个。

**易错点**

- 连续分配优先级高于零散分配。
- 同一排连续座位要按排从小到大、起点从小到大找。
- 每次输出本次分到的座位编号，不是输出所有已占座位。

**复杂度**

座位总数固定为 100。每次请求最多扫描常数个座位，时间复杂度 `O(n * 100)`，空间复杂度 `O(100)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<vector<int>> used(20, vector<int>(5, 0));

    while (n--) {
        int p;
        cin >> p;
        vector<int> ans;

        for (int row = 0; row < 20 && ans.empty(); ++row) {
            for (int start = 0; start + p <= 5; ++start) {
                bool ok = true;
                for (int col = start; col < start + p; ++col) {
                    ok &= !used[row][col];
                }
                if (!ok) continue;
                for (int col = start; col < start + p; ++col) {
                    used[row][col] = 1;
                    ans.push_back(row * 5 + col + 1);
                }
                break;
            }
        }

        if (ans.empty()) {
            for (int row = 0; row < 20 && (int)ans.size() < p; ++row) {
                for (int col = 0; col < 5 && (int)ans.size() < p; ++col) {
                    if (!used[row][col]) {
                        used[row][col] = 1;
                        ans.push_back(row * 5 + col + 1);
                    }
                }
            }
        }

        for (int i = 0; i < (int)ans.size(); ++i) {
            if (i) cout << ' ';
            cout << ans[i];
        }
        cout << '\n';
    }

    return 0;
}
```

### 第09次 T2 工资计算

**题意抓手**

已知某人的税后工资 `S`，根据题目给出的个税规则，求税前工资。个税只对超过 3500 的部分分段计税。

**知识点**

税前工资到税后工资是单调不减函数。分段税率会让直接逆推比较繁琐，但二分答案只需要能正向计算税后工资。这里二分区间取 `[0, 1000000]` 足够覆盖题目数据。

**解法**

写 `after_tax(salary)`，先算应纳税所得额 `taxable=max(0,salary-3500)`，再按题目税率表计算税额。二分最小的 `salary`，使 `after_tax(salary) >= S`。题目保证答案存在且为整数。

**易错点**

- 速算扣除数要和税率区间配套。
- 3500 以下不纳税，税后工资等于税前工资。
- 二分时找“最小满足条件”的税前工资，更新边界不能写反。

**复杂度**

税率表长度固定，二分次数固定在约 20 次以内。时间复杂度 `O(log M)`，`M` 为搜索上界；空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

double after_tax(int salary) {
    int taxable = max(0, salary - 3500);
    double tax = 0;
    if (taxable <= 1500) tax = taxable * 0.03;
    else if (taxable <= 4500) tax = taxable * 0.10 - 105;
    else if (taxable <= 9000) tax = taxable * 0.20 - 555;
    else if (taxable <= 35000) tax = taxable * 0.25 - 1005;
    else if (taxable <= 55000) tax = taxable * 0.30 - 2755;
    else if (taxable <= 80000) tax = taxable * 0.35 - 5505;
    else tax = taxable * 0.45 - 13505;
    return salary - tax;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int s;
    cin >> s;

    int left = 0, right = 1000000;
    while (left < right) {
        int mid = (left + right) / 2;
        if (after_tax(mid) + 1e-9 < s) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }

    cout << left << '\n';
    return 0;
}
```

### 第10次 T2 学生排队

**题意抓手**

队伍初始为 `1..n`。每条指令给出学生编号和移动量，正数表示向后移动，负数表示向前移动。所有操作依次生效，最后输出队伍顺序。

**知识点**

顺序表的删除和插入。`vector` 支持按位置删除和插入，虽然单次是 `O(n)`，但本题规模小，直接模拟最清楚。

**解法**

每次用 `find` 找到学生当前位置 `pos`，先 `erase` 删除该学生，再在 `pos + delta` 位置 `insert` 回去。题目保证移动合法，所以不需要额外裁剪边界。

**易错点**

- 先删再插，不能先插再删。
- `delta` 是相对于原位置的变化，删除后用 `pos + delta` 插入正好成立。
- 输出一行，编号之间一个空格。

**复杂度**

每次操作查找、删除、插入都是 `O(n)`，总时间复杂度 `O(mn)`，空间复杂度 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<int> q(n);
    iota(q.begin(), q.end(), 1);

    while (m--) {
        int id, delta;
        cin >> id >> delta;
        int pos = find(q.begin(), q.end(), id) - q.begin();
        q.erase(q.begin() + pos);
        q.insert(q.begin() + pos + delta, id);
    }

    for (int i = 0; i < n; ++i) {
        if (i) cout << ' ';
        cout << q[i];
    }
    cout << '\n';

    return 0;
}
```
