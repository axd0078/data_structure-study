## 学习内容

Day 020 是前 20 天的一个小复盘，也继续补两道早期 T2。**Z字形扫描**训练二维矩阵的遍历顺序建模；**数字排序**训练频次统计和自定义排序。一个重点在“按对角线输出”，一个重点在“排序规则不要写反”。

Z 字形扫描可以按副对角线分组。每个格子 `(i,j)` 都属于编号 `i+j` 的对角线。对角线编号从 `0` 到 `2n-2`。方向交替：

- `i+j` 为偶数时，从下往上输出；
- `i+j` 为奇数时，从上往下输出。

数字排序先统计每个数出现次数，再按两个关键字排序：

```text
出现次数多的在前
次数相同，数值小的在前
```

因为数值范围只有 `0..1000`，用计数数组比 `map` 更直接。

## 题解

### 第03次 T2 Z字形扫描

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

输入一个 `n x n` 矩阵，按照题图所示的 Z 字形顺序输出所有元素。这个顺序本质上是按 `i+j` 相同的副对角线依次输出，并且每条对角线方向交替。

**知识点**

对任意格子 `(i,j)`，副对角线编号为：

```text
sum = i + j
```

固定 `sum` 后，合法格子满足：

```text
0 <= i < n
0 <= sum - i < n
```

如果按 `i` 从小到大收集，得到的是从上到下的顺序。题目要求偶数对角线反过来输出，奇数对角线保持这个顺序。

**解法**

1. 读入矩阵。
2. 枚举 `sum = 0..2n-2`。
3. 收集当前对角线上的所有坐标。
4. 如果 `sum` 为偶数，反转当前坐标列表。
5. 依次输出这些坐标对应的元素。

**易错点**

- 输出是一行，元素之间一个空格，行末换行。
- `sum` 最大是 `(n-1)+(n-1)=2n-2`。
- 偶数对角线和奇数对角线方向不要写反；可以用样例前几项校验。
- `n` 可到 500，矩阵最多 25 万个元素，存矩阵没有压力。

**复杂度**

时间复杂度 `O(n^2)`，空间复杂度 `O(n^2)` 保存矩阵；每条对角线临时坐标最多 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<vector<int>> a(n, vector<int>(n));
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            cin >> a[i][j];
        }
    }

    bool first = true;
    for (int sum = 0; sum <= 2 * n - 2; ++sum) {
        vector<pair<int, int>> cells;
        for (int i = 0; i < n; ++i) {
            int j = sum - i;
            if (0 <= j && j < n) {
                cells.push_back({i, j});
            }
        }

        if (sum % 2 == 0) {
            reverse(cells.begin(), cells.end());
        }

        for (auto [i, j] : cells) {
            if (!first) {
                cout << ' ';
            }
            first = false;
            cout << a[i][j];
        }
    }

    cout << '\n';
    return 0;
}
```

### 第04次 T2 数字排序

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

输入 `n` 个非负整数，统计每个整数出现次数，并按规则输出：

1. 出现次数多的先输出；
2. 次数相同，数值小的先输出。

**知识点**

数值范围不超过 1000，直接用 `count[value]` 统计。统计后只把出现过的数放入数组排序。排序比较器要先比较次数，再比较数值：

```text
count 降序
value 升序
```

**解法**

1. 建立长度 1001 的计数数组。
2. 读入每个数并计数。
3. 把 `count[value] > 0` 的数转成 `(value,count)`。
4. 排序：
   - 次数不同，次数大的在前；
   - 次数相同，值小的在前。
5. 输出排序后的每一项。

**易错点**

- 不是按数字大小排序，而是先按出现次数排序。
- 次数相同才按数字升序。
- 输入数字可能是 0。
- 输出每个不同数字一行，不输出没出现过的数字。

**复杂度**

设值域大小 `V=1001`。时间复杂度 `O(n + V log V)`，额外空间复杂度 `O(V)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> count(1001, 0);
    for (int i = 0; i < n; ++i) {
        int x;
        cin >> x;
        ++count[x];
    }

    vector<pair<int, int>> items;
    for (int value = 0; value <= 1000; ++value) {
        if (count[value] > 0) {
            items.push_back({value, count[value]});
        }
    }

    sort(items.begin(), items.end(), [](const auto& lhs, const auto& rhs) {
        if (lhs.second != rhs.second) {
            return lhs.second > rhs.second;
        }
        return lhs.first < rhs.first;
    });

    for (auto [value, times] : items) {
        cout << value << ' ' << times << '\n';
    }

    return 0;
}
```
