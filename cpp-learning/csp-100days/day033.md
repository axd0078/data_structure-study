# Day 033：矩阵视图维护与函数倍增

## 学习内容

今天两题都是“多次操作后查询”，关键是维护压缩状态。

矩阵重塑（其二）里，重塑只改变当前行列数，不改变当前行优先展开序列；转置才会真正改变元素顺序。由于总元素数不超过 `10^4`，转置次数不超过 100，直接在转置时重排一维数组是最稳的做法。查询时用当前列数计算线性下标。

字符串变换是函数迭代。字符集合只有 63 种，`k` 可到 `1e9`。可以对每个字符做二进制倍增：`jump[b][c]` 表示字符 `c` 经过 `2^b` 次变换后的字符。每次查询按 `k` 的二进制位套用即可。

## 题解

### 第34次 T2 矩阵重塑（其二）

**题意抓手**

给定初始矩阵，支持三种操作：重塑为 `p x q`、转置、查询当前矩阵 `(i,j)` 的元素。

**知识点**

矩阵行优先展开。无论当前形状如何，都可以用一维数组保存当前矩阵的行优先序列。重塑只是改变解释这个一维数组的行列数；转置则要把 `old[i][j]` 放到新矩阵 `new[j][i]`。

**解法**

维护 `flat`、`rows`、`cols`。重塑时更新 `rows,cols`。转置时新建数组 `transposed[j*rows+i]=flat[i*cols+j]`，然后交换行列。查询时输出 `flat[i*cols+j]`。

**易错点**

- 重塑不改变一维序列内容。
- 转置后行列数要交换。
- 查询下标从 0 开始。

**复杂度**

重塑和查询 `O(1)`，每次转置 `O(nm)`。转置次数不超过 100，总复杂度可过。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int rows, cols, t;
    cin >> rows >> cols >> t;
    int total = rows * cols;
    vector<int> flat(total);
    for (int& value : flat) cin >> value;

    while (t--) {
        int op, a, b;
        cin >> op >> a >> b;
        if (op == 1) {
            rows = a;
            cols = b;
        } else if (op == 2) {
            vector<int> transposed(total);
            for (int i = 0; i < rows; ++i) {
                for (int j = 0; j < cols; ++j) {
                    transposed[j * rows + i] = flat[i * cols + j];
                }
            }
            flat.swap(transposed);
            swap(rows, cols);
        } else {
            cout << flat[a * cols + b] << '\n';
        }
    }

    return 0;
}
```

### 第35次 T2 字符串变换

**题意抓手**

给定初始字符串 `s` 和字符替换函数 `f`。每次变换会把字符串中每个字符 `ch` 替换成 `f(ch)`。对多个查询 `k`，输出变换 `k` 次后的字符串。

**知识点**

函数迭代倍增。字符种类少，可以预处理每个字符经过 `2^0,2^1,...` 次变换后的结果。`k<=1e9`，31 层足够。

**解法**

先建立 `jump[0][ch]=f(ch)`，未定义字符映射到自己。递推：

```text
jump[b][ch] = jump[b-1][ jump[b-1][ch] ]
```

回答查询时，对字符串中每个字符按照 `k` 的二进制位依次跳转。

**易错点**

- 输入字符串和映射对用 `#` 包裹，内部可能有空格，不能简单用 `cin >>` 读字符串后丢失空格。
- 未定义字符保持不变。
- 每个查询都从初始字符串开始，不是接着上一次查询继续变换。

**复杂度**

预处理 `O(字符集 * log k)`，每次查询 `O(|s| log k)`，空间复杂度 `O(字符集 * log k)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string line;
    getline(cin, line);
    string initial = line.substr(1, line.size() - 2);

    int n;
    cin >> n;
    getline(cin, line);

    vector<array<int, 128>> jump(31);
    for (int ch = 0; ch < 128; ++ch) jump[0][ch] = ch;

    for (int i = 0; i < n; ++i) {
        getline(cin, line);
        jump[0][(unsigned char)line[1]] = (unsigned char)line[2];
    }

    for (int bit = 1; bit < 31; ++bit) {
        for (int ch = 0; ch < 128; ++ch) {
            jump[bit][ch] = jump[bit - 1][jump[bit - 1][ch]];
        }
    }

    int m;
    cin >> m;
    while (m--) {
        int k;
        cin >> k;
        string result = initial;
        for (char& ch : result) {
            int cur = (unsigned char)ch;
            int value = k;
            int bit = 0;
            while (value > 0) {
                if (value & 1) cur = jump[bit][cur];
                value >>= 1;
                ++bit;
            }
            ch = (char)cur;
        }
        cout << '#' << result << '#' << '\n';
    }

    return 0;
}
```
