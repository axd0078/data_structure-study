# Day 032：质因数分解与集合相似度

## 学习内容

今天两题分别练数论分解和字符串集合。

因子化简需要把 `n` 分解为素因子幂，只保留指数不少于 `k` 的整项。`n <= 1e10`，试除到 `sqrt(n)` 最多约 `1e5`，查询数不超过 10，直接试除足够稳。注意剩下的大素数指数只能是 1，而 `k>1`，因此不会被保留。

相似度计算要求忽略大小写，把文章变成单词集合，再计算交集和并集大小。`set` 或 `unordered_set` 都能做；由于单词长度短，数量 `1e4`，实现上更重视小写转换和去重。

## 题解

### 第32次 T2 因子化简

**题意抓手**

对每个查询给定 `n,k`。若 `n` 的素因子分解中某个素因子的指数小于 `k`，就删掉这一整项；指数不少于 `k` 的项保留。输出保留项乘积，若没有保留项输出 1。

**知识点**

质因数分解。对 `p` 从 2 开始试除，统计 `p` 的指数。每处理完一个素因子，如果指数 `count>=k`，就把 `p^count` 乘入答案。

**解法**

对每个查询独立分解。循环条件为 `p*p<=n`，其中 `n` 会在除因子时不断变小。2 单独处理后，后续只枚举奇数能略微减少时间。题目保证 `k>1`，所以最后剩余的大素数指数为 1，不会被保留。

**易错点**

- 保留的是 `p^count` 整项，不是只保留 `p^k`。
- `n` 可到 `1e10`，使用 `long long`。
- 如果没有任何保留因子，答案是 1。

**复杂度**

每次查询试除到 `sqrt(n)`，时间复杂度 `O(sqrt(n))`，空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int q;
    cin >> q;
    while (q--) {
        long long n;
        int k;
        cin >> n >> k;

        long long answer = 1;
        for (long long p = 2; p * p <= n; p += (p == 2 ? 1 : 2)) {
            int count = 0;
            while (n % p == 0) {
                n /= p;
                ++count;
            }
            if (count >= k) {
                while (count--) answer *= p;
            }
        }

        cout << answer << '\n';
    }

    return 0;
}
```

### 第33次 T2 相似度计算

**题意抓手**

给两篇文章的单词序列，忽略大小写后分别去重为集合 `A` 和 `B`。输出 `|A∩B|` 和 `|A∪B|`。

**知识点**

字符串归一化和集合操作。大小写忽略意味着所有单词读入后先转小写，再插入集合。

**解法**

读第一篇文章，转小写后插入 `set<string> a`；第二篇同理。枚举较方便的一个集合，统计有多少词也在另一个集合中，这是交集大小。并集大小可以用 `|A| + |B| - |A∩B|` 得到。

**易错点**

- `The`、`the`、`THE` 是同一个词。
- 同一篇文章内部重复词只算一次。
- 输出两行，不需要输出相似度小数。

**复杂度**

设总单词数为 `N`，使用 `set` 的时间复杂度 `O(N log N)`，空间复杂度 `O(N)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

string lower_word(string s) {
    for (char& ch : s) {
        ch = (char)tolower((unsigned char)ch);
    }
    return s;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    set<string> a, b;
    for (int i = 0; i < n; ++i) {
        string word;
        cin >> word;
        a.insert(lower_word(word));
    }
    for (int i = 0; i < m; ++i) {
        string word;
        cin >> word;
        b.insert(lower_word(word));
    }

    int intersection_size = 0;
    for (const string& word : a) {
        intersection_size += b.count(word);
    }
    int union_size = (int)a.size() + (int)b.size() - intersection_size;

    cout << intersection_size << '\n' << union_size << '\n';
    return 0;
}
```
