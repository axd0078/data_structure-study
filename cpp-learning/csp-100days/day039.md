# Day 039：模板替换与网格涂色

## 学习内容

今天的两题都是模拟，但重点不同。

《模板生成系统》要抓住“合法标记已经保证”的条件。标记一定形如 `{{ VAR }}`，变量不存在就替换为空，变量值中即使包含新的标记也不递归处理。实现时逐字符扫描模板行，遇到 `{{ ` 就找到后面的 ` }}`，取出变量名查表替换。

《画图》是二维坐标和 BFS 填充。输入坐标以左下角为 `(0,0)`，而字符串输出按从上到下的行输出，所以需要一个 `row(y)=n-1-y` 的转换。填充时边界是线段字符 `- | +`，不是“当前填充字符”；如果某个格子原本已经是要填充的字符，也仍然应该作为连通区域继续扩展，因此参考代码使用 `seen` 数组防止重复访问。

## 题解

### 第05次 T3 模板生成系统

**题意抓手**

给定模板文本和若干变量，模板中的 `{{ VAR }}` 要替换成变量值；未定义变量替换为空；变量值不再递归替换。

**知识点**

字符串扫描、哈希表。由于题目保证标记合法，不需要写复杂的语法分析器。

**解法**

读入模板行后，读入变量定义。变量定义的值被双引号包住，可能包含空格，所以要整行读取，再用第一个和最后一个双引号截取值。渲染模板时，从左到右扫描：普通字符直接输出；遇到 `{{ `，找到最近的 ` }}`，取中间变量名查表，替换后跳过整个标记。

**易错点**

- `m` 行模板可能包含空格，必须用 `getline`。
- 变量值可能包含空格，不能用 `cin >> value` 直接读。
- 替换不递归，变量值里的 `{{ x }}` 原样输出。

**复杂度**

每个字符最多扫描一次，时间复杂度 `O(模板总长度+变量总长度)`，空间复杂度为模板和变量表大小。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

string render_line(const string& line, const unordered_map<string, string>& vars) {
    string out;
    for (size_t i = 0; i < line.size();) {
        if (i + 3 < line.size() && line[i] == '{' && line[i + 1] == '{' && line[i + 2] == ' ') {
            size_t end = line.find(" }}", i + 3);
            string name = line.substr(i + 3, end - (i + 3));
            auto it = vars.find(name);
            if (it != vars.end()) {
                out += it->second;
            }
            i = end + 3;
        } else {
            out.push_back(line[i]);
            ++i;
        }
    }
    return out;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int m, n;
    cin >> m >> n;
    string line;
    getline(cin, line);
    vector<string> tmpl(m);
    for (int i = 0; i < m; ++i) {
        getline(cin, tmpl[i]);
    }

    unordered_map<string, string> vars;
    for (int i = 0; i < n; ++i) {
        getline(cin, line);
        size_t space = line.find(' ');
        string name = line.substr(0, space);
        size_t first_quote = line.find('"', space);
        size_t last_quote = line.rfind('"');
        vars[name] = line.substr(first_quote + 1, last_quote - first_quote - 1);
    }

    for (const string& row : tmpl) {
        cout << render_line(row, vars) << '\n';
    }
    return 0;
}
```

### 第06次 T3 画图

**题意抓手**

维护一个 `m*n` 的字符画布，支持画水平/竖直线段和从某点开始的四连通填充，最后输出画布。

**知识点**

二维数组、坐标系转换、BFS Flood Fill。线段叠加时，水平线与竖直线相交要变成 `+`。

**解法**

画布内部按输出顺序存储，即第 0 行是最上方；输入的 `y` 用 `n-1-y` 转换成数组行号。画线时枚举端点之间所有格子：竖线遇到 `-` 或 `+` 变 `+`，否则写 `|`；横线遇到 `|` 或 `+` 变 `+`，否则写 `-`。填充时从起点 BFS，只要不是线段字符就可以进入，并写成目标字符。

**易错点**

- 填充不能把已有的同字符格子当作访问标记，否则会在同字符区域前停止扩展。
- 起点保证不在线段上，但可能已经被之前填过色。
- 输出顺序是从上到下，不是从 `y=0` 到 `y=n-1`。

**复杂度**

每次画线最多 `O(max(m,n))`，每次填充最多访问整张画布 `O(mn)`。`m,n,q<=100`，总复杂度充足。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int m, n, q;
    cin >> m >> n >> q;
    vector<string> canvas(n, string(m, '.'));

    auto row = [&](int y) { return n - 1 - y; };

    while (q--) {
        int op;
        cin >> op;
        if (op == 0) {
            int x1, y1, x2, y2;
            cin >> x1 >> y1 >> x2 >> y2;
            if (x1 == x2) {
                if (y1 > y2) swap(y1, y2);
                for (int y = y1; y <= y2; ++y) {
                    char& cell = canvas[row(y)][x1];
                    cell = (cell == '-' || cell == '+') ? '+' : '|';
                }
            } else {
                if (x1 > x2) swap(x1, x2);
                for (int x = x1; x <= x2; ++x) {
                    char& cell = canvas[row(y1)][x];
                    cell = (cell == '|' || cell == '+') ? '+' : '-';
                }
            }
        } else {
            int x, y;
            char c;
            cin >> x >> y >> c;
            int sr = row(y), sc = x;
            if (canvas[sr][sc] == '-' || canvas[sr][sc] == '|' || canvas[sr][sc] == '+') continue;
            queue<pair<int, int>> que;
            vector<vector<int>> seen(n, vector<int>(m, 0));
            que.push({sr, sc});
            seen[sr][sc] = 1;
            canvas[sr][sc] = c;
            const int dr[4] = {1, -1, 0, 0};
            const int dc[4] = {0, 0, 1, -1};
            while (!que.empty()) {
                auto [r, col] = que.front();
                que.pop();
                for (int d = 0; d < 4; ++d) {
                    int nr = r + dr[d], nc = col + dc[d];
                    if (nr < 0 || nr >= n || nc < 0 || nc >= m) continue;
                    if (seen[nr][nc]) continue;
                    char ch = canvas[nr][nc];
                    if (ch == '-' || ch == '|' || ch == '+') continue;
                    seen[nr][nc] = 1;
                    canvas[nr][nc] = c;
                    que.push({nr, nc});
                }
            }
        }
    }

    for (const string& line : canvas) {
        cout << line << '\n';
    }
    return 0;
}
```
