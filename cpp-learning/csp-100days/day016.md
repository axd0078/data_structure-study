# Day 016：边界模拟与等差求和

## 学习内容

Day 016 的两道题都要先把题面动作压成稳定的循环或公式。**移动**是典型逐字符模拟：每条指令只改变一个方向，目标点越界就不移动。**数值积分**看起来要遍历区间内所有偶数点，但因为函数是二次函数、坐标点构成等差数列，可以用求和公式做到 `O(1)`。

移动题的关键是“先试探，再提交”：读到一条指令后先算临时位置 `(nx,ny)`，只有 `1 <= nx,ny <= n` 时才更新当前位置。这样边界处理不会污染原位置。

数值积分要计算：

```text
s = 2 * sum(x^2 + b*x + c)
```

其中 `x` 是 `[l,r]` 内的偶数。设第一个偶数为 `start`，共有 `cnt` 项，项为 `start, start+2, ...`。需要用到 `sum x` 和 `sum x^2`。范围很小，循环也能过；但这里直接练习等差数列求和，复杂度最低。

今天的最低复杂度目标：

- 移动：`O(总指令长度)` 时间，`O(1)` 额外空间。
- 数值积分：`O(1)` 时间，`O(1)` 空间。

## 题解

### 第36次 T1 移动

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

有一个 `n x n` 方格，机器人起点为 `(x,y)`。每个查询给一串由 `f,b,l,r` 组成的指令：

- `f`：`y+1`
- `b`：`y-1`
- `l`：`x-1`
- `r`：`x+1`

如果下一步会走出 `[1,n] x [1,n]`，本步无效，机器人保持原位置。每个查询互相独立。

**知识点**

这类边界模拟不要直接修改原坐标。更稳的写法是先复制成 `nx,ny`，根据指令修改临时坐标，再判断临时坐标是否合法。合法才赋值回 `x,y`。

**解法**

1. 读入 `n,k`。
2. 对每个查询读入 `x,y,ops`。
3. 扫描 `ops`：
   - 用 `nx,ny` 记录试探位置。
   - 按字符修改一个方向。
   - 如果试探位置在边界内，更新 `x=nx,y=ny`。
4. 输出最终位置。

**易错点**

- 每个查询从自己的起点开始，不继承上一个查询的位置。
- 越界时只忽略当前指令，不是终止整串指令。
- `f/b` 改的是 `y`，`l/r` 改的是 `x`。

**复杂度**

时间复杂度 `O(总指令长度)`，额外空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;

    while (k--) {
        int x, y;
        string ops;
        cin >> x >> y >> ops;

        for (char op : ops) {
            int nx = x;
            int ny = y;
            if (op == 'f') {
                ++ny;
            } else if (op == 'b') {
                --ny;
            } else if (op == 'l') {
                --nx;
            } else if (op == 'r') {
                ++nx;
            }

            if (1 <= nx && nx <= n && 1 <= ny && ny <= n) {
                x = nx;
                y = ny;
            }
        }

        cout << x << ' ' << y << '\n';
    }

    return 0;
}
```

### 第37次 T1 数值积分

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

给定二次函数：

```text
f(x) = x^2 + b*x + c
```

只取闭区间 `[l,r]` 内的偶数坐标点，把这些点的函数值求和后乘以 2，输出整数结果。

**知识点**

偶数坐标点构成公差为 2 的等差数列。设：

```text
x_t = start + 2t,  t = 0..cnt-1
```

则：

```text
sum x_t = cnt * (start + last) / 2
sum x_t^2 = cnt*start^2 + 4*start*sum(t) + 4*sum(t^2)
```

其中：

```text
sum(t) = cnt*(cnt-1)/2
sum(t^2) = cnt*(cnt-1)*(2*cnt-1)/6
```

**解法**

1. 第一个偶数 `start = l`，如果 `l` 是奇数则为 `l+1`。
2. 项数 `cnt = (r-start)/2 + 1`。
3. 用公式计算 `sum_x` 和 `sum_x2`。
4. 答案为 `2 * (sum_x2 + b*sum_x + c*cnt)`。

**易错点**

- 题目保证 `l+2 <= r`，但仍建议写 `start > r` 的保护。
- `x^2`、求和和乘法统一用 `long long`。
- 只取偶数坐标点，不是从 `l` 每次加 2；如果 `l` 是奇数，要先跳到 `l+1`。

**复杂度**

时间复杂度 `O(1)`，额外空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long b, c, l, r;
    cin >> b >> c >> l >> r;

    long long start = (l % 2 == 0 ? l : l + 1);
    if (start > r) {
        cout << 0 << '\n';
        return 0;
    }

    long long cnt = (r - start) / 2 + 1;
    long long last = start + 2 * (cnt - 1);
    long long sum_t = cnt * (cnt - 1) / 2;
    long long sum_t2 = cnt * (cnt - 1) * (2 * cnt - 1) / 6;
    long long sum_x = cnt * (start + last) / 2;
    long long sum_x2 = cnt * start * start + 4 * start * sum_t + 4 * sum_t2;

    cout << 2 * (sum_x2 + b * sum_x + c * cnt) << '\n';
    return 0;
}
```
