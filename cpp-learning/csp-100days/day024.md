# Day 024：区间重叠、信号灯时间轴与表达式求值

## 学习内容

今天的三题把“模拟”推进到更清晰的模型层：区间、周期、表达式。

买菜是两个有序区间集合求交集总长度。每个人的装车时间段已经按开始时间递增给出，用双指针比较当前两个区间即可。交集长度是 `max(0, min(r1,r2) - max(l1,l2))`，然后让结束更早的那个区间向后移动。

小明放学是周期时间轴。红绿灯的完整周期为 `r + g + y`，可以把任意时刻映射到周期中的位置。题目给出的 `k,t` 是小明到达路口时看到的灯和剩余时间，但他之前已经走了 `total` 秒，所以要在这个基础上继续推进 `total`，判断实际到达时还要等多久。

二十四点是无括号表达式求值。只有 4 个数字和 3 个运算符，表达式长度固定为 7。按四则运算优先级，可以先处理乘除，把加减项压缩成若干数字，再从左到右处理加减。

## 题解

### 第14次 T2 买菜

**题意抓手**

两个人各有 `n` 个在店里的时间区间，要求他们同时在店里的总时间。区间按时间顺序给出。

**知识点**

有序区间交集。两个当前区间 `[l1,r1)` 和 `[l2,r2)` 的重叠长度为 `min(r1,r2)-max(l1,l2)`，如果结果为负则没有重叠。

**解法**

用两个指针 `i,j` 分别指向两人的当前区间。计算当前交集并累计。谁的右端点更小，谁就不可能再和对方当前区间产生更多交集，因此移动那个指针。

**易错点**

- 区间是左闭右开，长度是 `right-left`。
- 不要用离散数组暴力标记大时间轴，双指针更直接。
- 当两个右端点相等时，移动任意一个都可以，这份代码移动第二个。

**复杂度**

每个区间最多被指针经过一次，时间复杂度 `O(n)`，空间复杂度 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<pair<int, int>> a(n), b(n);
    for (int i = 0; i < n; ++i) cin >> a[i].first >> a[i].second;
    for (int i = 0; i < n; ++i) cin >> b[i].first >> b[i].second;

    int i = 0, j = 0, ans = 0;
    while (i < n && j < n) {
        int left = max(a[i].first, b[j].first);
        int right = min(a[i].second, b[j].second);
        if (left < right) ans += right - left;

        if (a[i].second < b[j].second) ++i;
        else ++j;
    }

    cout << ans << '\n';
    return 0;
}
```

### 第15次 T2 小明放学

**题意抓手**

小明按顺序经过若干路段或红绿灯。`k=0` 是路段，直接花 `t` 秒；`k=1/2/3` 分别表示到达时看到红/黄/绿灯，且该颜色还剩 `t` 秒。由于小明之前已经走了若干秒，当前看到的灯会随时间变化，要计算实际到达时是否需要等待。

**知识点**

周期模拟。把一个灯的周期看成红 `[0,r)`、绿 `[r,r+g)`、黄 `[r+g,r+g+y)`。根据题目给的当前颜色和剩余时间，先还原到周期中的起始位置，再加上已经过去的总时间取模。

**解法**

维护总用时 `total`。遇到路段直接累加。遇到灯时，先用 `kind, remain` 还原观测时刻在周期中的位置 `start`：

- 红灯剩 `remain`：`start = r - remain`；
- 绿灯剩 `remain`：`start = r + (g - remain)`；
- 黄灯剩 `remain`：`start = r + g + (y - remain)`。

实际到达位置为 `(start + total) % cycle`。如果在红灯区，等待到红灯结束；如果在绿灯区，不等；如果在黄灯区，等待黄灯剩余加下一轮红灯。

**易错点**

- 周期顺序是红、绿、黄。
- 黄灯不能通行，要等黄灯结束再等一整段红灯。
- 总时间可能很大，使用 `long long`。

**复杂度**

每条记录处理常数次，时间复杂度 `O(n)`，空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long r, y, g;
    int n;
    cin >> r >> y >> g >> n;
    long long cycle = r + y + g;
    long long total = 0;

    auto wait_time = [&](int kind, long long remain, long long elapsed) {
        long long start;
        if (kind == 1) {
            start = r - remain;
        } else if (kind == 2) {
            start = r + g + (y - remain);
        } else {
            start = r + (g - remain);
        }

        long long cur = (start + elapsed) % cycle;
        if (cur < r) return r - cur;
        if (cur < r + g) return 0LL;
        return cycle - cur;
    };

    for (int i = 0; i < n; ++i) {
        int k;
        long long t;
        cin >> k >> t;
        if (k == 0) total += t;
        else total += wait_time(k, t, total);
    }

    cout << total << '\n';
    return 0;
}
```

### 第16次 T2 二十四点

**题意抓手**

给定若干个长度为 7 的表达式，每个表达式由 4 个 1-9 的数字和 3 个运算符组成，没有括号。按四则运算规则和整数除法判断结果是否为 24。

**知识点**

表达式求值的优先级处理。由于没有括号，可以第一遍处理乘除，把它们合并进当前数字；遇到加减时，把运算符和新数字保存下来。第二遍再处理加减。

**解法**

初始化 `nums` 为第一个数字。每读一个运算符和下一个数字：

- 如果是 `x`，就让 `nums.back() *= value`；
- 如果是 `/`，就让 `nums.back() /= value`；
- 如果是 `+` 或 `-`，保存运算符，并把 `value` 加入 `nums`。

最后从 `nums[0]` 开始按保存的加减符号计算结果。

**易错点**

- 乘号是小写字母 `x`，不是 `*`。
- 除法是整数除法。
- 表达式中的数字都是单个字符，不需要解析多位数。

**复杂度**

每个表达式长度固定为 7，时间复杂度 `O(1)`；处理 `n` 个表达式为 `O(n)`。空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int eval_expr(const string& s) {
    vector<int> nums;
    vector<char> ops;
    nums.push_back(s[0] - '0');

    for (int i = 1; i < (int)s.size(); i += 2) {
        char op = s[i];
        int value = s[i + 1] - '0';
        if (op == 'x') {
            nums.back() *= value;
        } else if (op == '/') {
            nums.back() /= value;
        } else {
            ops.push_back(op);
            nums.push_back(value);
        }
    }

    int result = nums[0];
    for (int i = 0; i < (int)ops.size(); ++i) {
        if (ops[i] == '+') result += nums[i + 1];
        else result -= nums[i + 1];
    }
    return result;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    while (n--) {
        string expr;
        cin >> expr;
        cout << (eval_expr(expr) == 24 ? "Yes" : "No") << '\n';
    }

    return 0;
}
```
