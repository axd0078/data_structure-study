## 学习内容

今天两题训练“把题意中的规则变成可维护的状态”。

《集合竞价》不要用浮点数处理价格，价格精确到两位小数，可以统一转成“分”。开盘价只需要在出现过的报价上评估：当候选价 `p` 固定时，能成交的买单是所有报价 `>=p` 的买量，能成交的卖单是所有报价 `<=p` 的卖量，成交量取两者最小。按价格排序后，买量可以从高到低做后缀和，卖量可以从低到高做前缀和，评估每个候选价即可。

《节日》是日期题，关键是统一星期编号。题面给出 1850-01-01 是星期二，且 `c=1..7` 表示周一到周日。只要能求出某年某月 1 日是星期几，就能算出这个月第 `b` 个星期 `c` 是哪一天。

## 题解

### 第03次 T3 集合竞价

**题意抓手**

输入若干买单、卖单和撤销记录。忽略被撤销的记录后，选择一个开盘价，使成交量最大；若多个价格成交量相同，输出最高价格。

**知识点**

离散候选价、前缀和、后缀和。价格转整数避免浮点误差，股数和成交量用 `long long`。

**解法**

先读入所有记录，再根据 `cancel i` 标记被撤销的记录和撤销记录本身。对剩余订单按价格聚合买量、卖量，并收集所有出现过的价格。价格从高到低扫描，得到 `buy_ge[p]`；价格从低到高扫描，累计 `sell_le[p]`，此时 `min(buy_ge[p], sell_le[p])` 就是以 `p` 开盘的成交量。更新答案时先比成交量，再比价格。

**易错点**

- `cancel i` 的 `i` 是输入行号，从 1 开始。
- 撤销记录本身不是订单，不能参与候选价。
- 价格输出必须保留两位小数。

**复杂度**

设有效价格数为 `U`，记录数为 `N`。排序和聚合为 `O(N log U)`，扫描为 `O(U)`，空间复杂度 `O(U+N)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Order {
    string side;
    int price;
    long long amount;
};

int parse_price(const string& s) {
    int dot = (int)s.find('.');
    int whole = stoi(s.substr(0, dot));
    int cents = stoi(s.substr(dot + 1));
    return whole * 100 + cents;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    vector<vector<string>> records;
    string op;
    while (cin >> op) {
        if (op == "cancel") {
            string index;
            cin >> index;
            records.push_back({op, index});
        } else {
            string price, amount;
            cin >> price >> amount;
            records.push_back({op, price, amount});
        }
    }

    vector<int> canceled(records.size());
    for (int i = 0; i < (int)records.size(); ++i) {
        if (records[i][0] == "cancel") {
            int target = stoi(records[i][1]) - 1;
            canceled[target] = true;
            canceled[i] = true;
        }
    }

    map<int, long long> buy_at_price;
    map<int, long long> sell_at_price;
    set<int> prices;
    for (int i = 0; i < (int)records.size(); ++i) {
        if (canceled[i] || records[i][0] == "cancel") {
            continue;
        }
        Order order{records[i][0], parse_price(records[i][1]), stoll(records[i][2])};
        if (order.side == "buy") {
            buy_at_price[order.price] += order.amount;
        } else {
            sell_at_price[order.price] += order.amount;
        }
        prices.insert(order.price);
    }

    map<int, long long> buy_ge;
    long long buy_sum = 0;
    for (auto it = prices.rbegin(); it != prices.rend(); ++it) {
        buy_sum += buy_at_price[*it];
        buy_ge[*it] = buy_sum;
    }

    int best_price = 0;
    long long best_volume = 0;
    long long sell_sum = 0;
    for (int price : prices) {
        sell_sum += sell_at_price[price];
        long long volume = min(buy_ge[price], sell_sum);
        if (volume > best_volume || (volume == best_volume && price > best_price)) {
            best_volume = volume;
            best_price = price;
        }
    }

    cout << best_price / 100 << '.' << setw(2) << setfill('0') << best_price % 100
         << ' ' << best_volume << '\n';
    return 0;
}
```

### 第04次 T3 节日

**题意抓手**

输出从 `y1` 到 `y2` 每一年 `a` 月第 `b` 个星期 `c` 的日期；如果这个月没有第 `b` 个星期 `c`，输出 `none`。

**知识点**

闰年判断、月天数、星期偏移。已知 1850-01-01 是星期二，可以从这个基准累加天数。

**解法**

先写 `leap(year)` 和 `days_in_month(year, month)`。对某年某月 1 日，从 1850-01-01 累加到该月 1 日前的总天数，得到星期。若 1 日星期为 `first`，则第一个星期 `c` 的日期为 `1+(c-first+7)%7`，第 `b` 个再加 `(b-1)*7`。若超过本月天数则不存在。

**易错点**

- 星期编号是周一 `1` 到周日 `7`。
- 1900 不是闰年，2000 是闰年。
- 输出年月日要补足四位、两位。

**复杂度**

年份范围最多 201 年，直接累加也很小。对每个输出年份计算一次，时间复杂度可视为 `O((y2-y1+1)*201)`，空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

bool leap(int year) {
    return year % 400 == 0 || (year % 4 == 0 && year % 100 != 0);
}

int days_in_month(int year, int month) {
    static int days[] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    if (month == 2 && leap(year)) {
        return 29;
    }
    return days[month];
}

int weekday_of_first_day(int year, int month) {
    int weekday = 2;
    for (int y = 1850; y < year; ++y) {
        weekday = (weekday + (leap(y) ? 366 : 365)) % 7;
        if (weekday == 0) {
            weekday = 7;
        }
    }
    for (int m = 1; m < month; ++m) {
        weekday = (weekday + days_in_month(year, m)) % 7;
        if (weekday == 0) {
            weekday = 7;
        }
    }
    return weekday;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int month, nth, weekday, y1, y2;
    cin >> month >> nth >> weekday >> y1 >> y2;

    for (int year = y1; year <= y2; ++year) {
        int first = weekday_of_first_day(year, month);
        int day = 1 + (weekday - first + 7) % 7 + (nth - 1) * 7;
        if (day > days_in_month(year, month)) {
            cout << "none\n";
        } else {
            cout << setw(4) << setfill('0') << year << '/'
                 << setw(2) << setfill('0') << month << '/'
                 << setw(2) << setfill('0') << day << '\n';
        }
    }
    return 0;
}
```
