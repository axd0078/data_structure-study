# Day 011：浮点统计与混合进制

## 学习内容

Day 011 继续训练 T1 的保分能力，但题目开始从“照着输入做模拟”转向“先把数学定义压成一行公式”。今天的重点不是公式难，而是实现时要让数据类型、输出精度和读题顺序都稳定。

**归一化处理**要掌握均值、方差和标准化。给定数列 `a_1..a_n`，平均值为 `avg=sum(a_i)/n`，方差为 `D=sum((a_i-avg)^2)/n`，每个数输出 `(a_i-avg)/sqrt(D)`。这类题的关键是：平均值确定前不能输出答案，所以至少要保存原数组；浮点比较只要求误差范围，不需要输出和样例完全一样的小数位。

**如此编码**本质是混合进制拆数。题面给出：

```text
c_0 = 1
c_i = c_{i-1} * a_i
m = c_0*b_1 + c_1*b_2 + ... + c_{n-1}*b_n
0 <= b_i < a_i
```

这和十进制按位拆数字类似，只是每一位的“进制”不是固定的 10，而是 `a_i`。第 `i` 位可以直接用 `m % a_i` 得到，然后让 `m /= a_i`，继续处理下一位。

今天的最低复杂度目标：

- 归一化处理：`O(n)` 时间，`O(n)` 空间保存原始数列。
- 如此编码：`O(n)` 时间，`O(1)` 额外空间，边读边输出每一位。

## 题解

### 第26次 T1 归一化处理

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

输入 `n` 和 `n` 个整数，对每个整数做归一化。归一化后的值定义为：

```text
f(a_i) = (a_i - avg) / sqrt(D)
avg = sum(a_i) / n
D = sum((a_i - avg)^2) / n
```

题目保证 `D >= 1`，所以不用处理除以 0。输出每个 `f(a_i)`，允许绝对误差或相对误差不超过 `1e-4`。

**知识点**

均值和方差都要用浮点数计算。虽然输入是整数，但 `avg` 往往不是整数，如果用整数除法会直接丢掉小数部分。建议从读入开始就把数存进 `double` 或者读成整数后在求和、计算时转成 `double`。

这题需要两次扫描：第一次读入并求和，得到 `avg`；第二次根据 `avg` 求方差；最后再输出每个标准化值。可以把“求方差”和“输出”拆成两个循环，代码更直观。

**解法**

1. 读入 `n`。
2. 保存所有 `a_i`，同时累加 `sum`。
3. 计算 `mean = sum / n`。
4. 遍历数组，累加 `(a_i - mean)^2`，最后除以 `n` 得到方差。
5. 计算 `denominator = sqrt(variance)`。
6. 再遍历数组，输出 `(a_i - mean) / denominator`。

**易错点**

- `sum / n` 必须是浮点除法。
- 方差定义里除以的是 `n`，不是 `n-1`。
- 样例的小数位很多，但本题是误差判定，输出 `fixed << setprecision(12)` 足够。
- `sqrt` 需要包含 `<cmath>`，使用 `bits/stdc++.h` 时已经包含。

**复杂度**

时间复杂度 `O(n)`，空间复杂度 `O(n)`。由于所有答案都依赖最终均值和方差，保存原始数组是最直接稳定的做法。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<double> a(n);
    double sum = 0.0;
    for (double &value : a) {
        cin >> value;
        sum += value;
    }

    double mean = sum / n;
    double variance = 0.0;
    for (double value : a) {
        double diff = value - mean;
        variance += diff * diff;
    }
    variance /= n;

    double denominator = sqrt(variance);
    cout << fixed << setprecision(12);
    for (double value : a) {
        cout << (value - mean) / denominator << '\n';
    }

    return 0;
}
```

### 第27次 T1 如此编码

- 原题链接：见上方题目表。
- 当前状态：题解已补齐。

**题意抓手**

给定 `n`、`m` 和 `a_1..a_n`。题目定义：

```text
c_0 = 1
c_i = c_{i-1} * a_i
m = c_0*b_1 + c_1*b_2 + ... + c_{n-1}*b_n
```

要求输出 `b_1..b_n`。每个 `b_i` 的范围是 `0 <= b_i < a_i`。

**知识点**

因为 `c_i` 是前缀乘积，所以 `b_i` 就是混合进制下的第 `i` 位。对于当前的 `m`：

```text
b_i = m % a_i
m = m / a_i
```

这个过程和把十进制数字不断 `% 10`、`/ 10` 拆出个位、十位很像，只是每一位的进制不同。

**解法**

1. 读入 `n` 和 `m`。
2. 从 `i=1` 到 `n` 依次读入 `a_i`。
3. 当前位答案为 `m % a_i`。
4. 更新 `m /= a_i`。
5. 按空格分隔输出所有位。

**易错点**

- 不需要真的维护 `c_i`，否则容易把公式写复杂。
- `m` 和前缀乘积最大到 `1e9`，用 `long long` 更稳。
- 输出是一行，数字之间一个空格，行末换行。

**复杂度**

时间复杂度 `O(n)`，额外空间复杂度 `O(1)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    long long m;
    cin >> n >> m;

    for (int i = 0; i < n; ++i) {
        long long radix;
        cin >> radix;
        if (i) {
            cout << ' ';
        }
        cout << m % radix;
        m /= radix;
    }
    cout << '\n';

    return 0;
}
```
