# Day 023：事件排序与小规模模拟

## 学习内容

今天继续练 T2 模拟，但题面规则更依赖“事件顺序”和“同时发生”的处理。

公共钥匙盒是典型事件排序题。每次借钥匙产生两个事件：开始时借走，结束时归还。同一时刻有多个事件时，必须先还后借；多个归还同时发生时，按钥匙编号从小到大归还到最左边空位。因此排序关键字可以设计成 `(time, type, key)`，其中归还事件 `type=0`，借出事件 `type=1`。

游戏是约瑟夫环变形。孩子围成环报数，报到 `k` 的倍数或个位为 `k` 就退出。由于 `n` 不大，直接用 `vector` 删除当前位置即可。关键是删除后当前位置不前进，因为下一个孩子已经移动到了当前下标。

碰撞的小球是物理规则模拟。所有小球初始向右，每秒先移动，再处理撞墙和相遇。题目范围允许按秒模拟，注意同一秒里位置相同的两个球要同时反向。这个题比公式推导更考验更新顺序。

## 题解

### 第11次 T2 公共钥匙盒

**题意抓手**

钥匙盒初始第 `i` 个位置放钥匙 `i`。每位老师在开始时间借走某把钥匙，持续一段时间后归还。归还时放入最左边空位。求所有事件结束后的钥匙盒状态。

**知识点**

事件流模拟。借出和归还都转成事件，按时间排序；同一时间的处理顺序由题目规则决定：先还再借，归还按钥匙编号小的先处理。

**解法**

每条记录 `(w,s,c)` 生成借出事件 `(s,1,w)` 和归还事件 `(s+c,0,w)`。排序后顺序处理。借出时在盒子里找到钥匙 `w` 并置为 0；归还时找到最左边的 0 放入 `w`。

**易错点**

- 同一时刻一定先归还再借出。
- 同一时刻归还多把钥匙时按钥匙编号升序。
- 归还不是放回原位置，而是放到当前最左空位。

**复杂度**

事件数为 `2k`，排序 `O(k log k)`；每个事件线性扫描钥匙盒 `O(n)`，总时间复杂度 `O(k log k + kn)`，空间复杂度 `O(n+k)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Event {
    int time;
    int type;
    int key;
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;
    vector<Event> events;
    for (int i = 0; i < k; ++i) {
        int w, s, c;
        cin >> w >> s >> c;
        events.push_back({s, 1, w});
        events.push_back({s + c, 0, w});
    }

    sort(events.begin(), events.end(), [](const Event& a, const Event& b) {
        if (a.time != b.time) return a.time < b.time;
        if (a.type != b.type) return a.type < b.type;
        return a.key < b.key;
    });

    vector<int> box(n);
    iota(box.begin(), box.end(), 1);

    for (const auto& e : events) {
        if (e.type == 0) {
            int pos = find(box.begin(), box.end(), 0) - box.begin();
            box[pos] = e.key;
        } else {
            int pos = find(box.begin(), box.end(), e.key) - box.begin();
            box[pos] = 0;
        }
    }

    for (int i = 0; i < n; ++i) {
        if (i) cout << ' ';
        cout << box[i];
    }
    cout << '\n';

    return 0;
}
```

### 第12次 T2 游戏

**题意抓手**

`n` 个小朋友围成一圈报数，从 1 开始递增。报到的数字如果是 `k` 的倍数，或者个位是 `k`，当前小朋友退出。求最后剩下的小朋友编号。

**知识点**

环形序列删除。用数组保存当前还在圈中的小朋友，`pos` 表示当前报数的人。删除后，如果删除的是最后一个位置，`pos` 回到 0；否则 `pos` 不变。

**解法**

初始化 `children = [1,2,...,n]`，`pos=0`，`number=1`。循环直到只剩 1 人。每轮判断 `number` 是否触发退出，若退出则删除 `children[pos]`，否则 `pos` 移到下一个人。最后输出唯一元素。

**易错点**

- 条件是“`number % k == 0` 或 `number % 10 == k`”。
- 删除后不要再额外 `pos++`。
- `number` 每轮都递增，即使有人退出也要递增。

**复杂度**

`vector` 删除为 `O(n)`，最多删除 `n-1` 次，时间复杂度 `O(n^2)`；题目规模小可以通过。空间复杂度 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

bool should_leave(long long number, int k) {
    return number % k == 0 || number % 10 == k;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;
    vector<int> children(n);
    iota(children.begin(), children.end(), 1);

    int pos = 0;
    long long number = 1;
    while ((int)children.size() > 1) {
        if (should_leave(number, k)) {
            children.erase(children.begin() + pos);
            if (pos == (int)children.size()) pos = 0;
        } else {
            pos = (pos + 1) % children.size();
        }
        ++number;
    }

    cout << children[0] << '\n';
    return 0;
}
```

### 第13次 T2 碰撞的小球

**题意抓手**

一条线段长度为 `L`，`n` 个小球初始都向右，每秒移动 1 个单位。碰到端点或两个球相遇时反向。模拟 `t` 秒后输出每个小球的位置。

**知识点**

逐秒同步模拟。每秒先把所有球按当前方向移动，再处理撞墙，最后处理相同位置的小球碰撞。用 `dir[i]` 表示第 `i` 个球方向，`1` 为向右，`-1` 为向左。

**解法**

执行 `t` 轮。每轮先更新所有位置，如果位置为 `0` 或 `L` 就反转方向；然后枚举所有球对，如果两个球位置相同，就把两者方向都反转。题目数据范围下这种直接模拟足够稳定。

**易错点**

- 同一秒要先移动所有球，再判断碰撞。
- 输出顺序是输入中小球的顺序。
- 撞墙和相遇都只改变方向，位置本身已经在本秒移动后确定。

**复杂度**

每秒移动 `O(n)`，检测球对 `O(n^2)`，总时间复杂度 `O(tn^2)`，空间复杂度 `O(n)`。

**参考代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, l, t;
    cin >> n >> l >> t;
    vector<int> x(n), dir(n, 1);
    for (int i = 0; i < n; ++i) cin >> x[i];

    for (int sec = 0; sec < t; ++sec) {
        for (int i = 0; i < n; ++i) {
            x[i] += dir[i];
            if (x[i] == 0 || x[i] == l) {
                dir[i] = -dir[i];
            }
        }
        for (int i = 0; i < n; ++i) {
            for (int j = i + 1; j < n; ++j) {
                if (x[i] == x[j]) {
                    dir[i] = -dir[i];
                    dir[j] = -dir[j];
                }
            }
        }
    }

    for (int i = 0; i < n; ++i) {
        if (i) cout << ' ';
        cout << x[i];
    }
    cout << '\n';

    return 0;
}
```
